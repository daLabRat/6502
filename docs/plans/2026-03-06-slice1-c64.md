# Slice 1: C64 Plays a Game — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** A D64 disk image loads and runs correctly on the emulated C64, displayed with an optional CRT shader, with a separate debugger window showing C64 hardware state and save state support.

**Architecture:** Five phases — (A) IEC/1541 end-to-end verification, (B) CRT shader via wgpu post-process, (C) multi-window debugger via egui deferred viewports, (D) save states via serde/MessagePack, (E) ROM browser polish. Each phase is independently testable.

**Tech Stack:** Rust, egui 0.30, eframe 0.30 (wgpu backend), WGSL shaders, rmp-serde, bytemuck

### Task A2: Fix IEC issues found by tracing

This task is conditional — its content depends on what Task A1 reveals. Common issues to look for:

**A2a — Drive never receives byte correctly:**
Check the bit-receive loop at ROM $EA0B. Verify `drive_via1_port_b_input()` returns correct DATA IN (bit 0) and CLK IN (bit 2) values. Trace should show CLK toggling 8 times per byte.

File: `crates/c64/src/iec_bus.rs`
```rust
pub fn drive_via1_port_b_input(&self) -> u8 {
    // DATA IN: bit 0 = 1 when bus DATA is LOW (inverted)
    // CLK IN:  bit 2 = 1 when bus CLK is LOW (inverted)
    let data_in = if self.data() { 0x00 } else { 0x01 };
    let clk_in  = if self.clk()  { 0x00 } else { 0x04 };
    data_in | clk_in
}
```
Confirm `self.data()` returns `!(self.c64_data || self.drive_data)` and `self.clk()` returns `!(self.c64_clk || self.drive_clk)`.

**A2b — Drive finds no matching file:**
The drive compares the filename from the OPEN command with the directory. Verify D64 image parsing returns correct sector/track chains and filename matching.

File: `crates/c64/src/drive1541/mod.rs` — check `find_file()` or equivalent.

**A2c — CIA fast serial interfering:**
The C64 KERNAL may try fast serial (CIA2 serial port) before falling back to slow serial. If the drive doesn't support it, the C64 must time out first. This adds ~2 seconds but should self-correct.

```wgsl
// CRT post-process shader
// Uniforms: mode (0=sharp, 1=scanlines, 2=crt), src_size (vec2f)

struct Uniforms {
    src_size: vec2f,
    dst_size: vec2f,
    mode: u32,
    _pad: u32,
}

@group(0) @binding(0) var t_screen: texture_2d<f32>;
@group(0) @binding(1) var s_screen: sampler;
@group(0) @binding(2) var<uniform> u: Uniforms;

struct VertexOutput {
    @builtin(position) pos: vec4f,
    @location(0) uv: vec2f,
}

@vertex
fn vs_main(@builtin(vertex_index) vi: u32) -> VertexOutput {
    // Full-screen triangle (3 vertices cover NDC)
    var out: VertexOutput;
    let x = f32(i32(vi) * 2 - 1);
    let y = f32(1 - i32(vi & 1u) * 4);
    out.pos = vec4f(x, y, 0.0, 1.0);
    out.uv  = vec2f((x + 1.0) * 0.5, (1.0 - y) * 0.5);
    return out;
}

@fragment
fn fs_main(in: VertexOutput) -> @location(0) vec4f {
    let uv = in.uv;

    if u.mode == 0u {
        // Sharp: nearest-neighbor (sampler handles this)
        return textureSample(t_screen, s_screen, uv);
    }

    // Scanlines (mode 1) and CRT (mode 2) share the base sample
    var col = textureSample(t_screen, s_screen, uv).rgb;

    if u.mode >= 1u {
        // Scanline darkening: every other output pixel row
        let screen_y = in.pos.y;
        let scanline = sin(screen_y * 3.14159 * (u.src_size.y / u.dst_size.y));
        let sl_factor = 0.75 + 0.25 * scanline * scanline;
        col *= sl_factor;

        // Phosphor glow: slight blur in x
        let ox = 1.0 / u.src_size.x;
        let left  = textureSample(t_screen, s_screen, vec2f(uv.x - ox, uv.y)).rgb;
        let right = textureSample(t_screen, s_screen, vec2f(uv.x + ox, uv.y)).rgb;
        col = col * 0.8 + left * 0.1 + right * 0.1;
    }

    if u.mode == 2u {
        // Barrel distortion
        var barrel_uv = uv * 2.0 - 1.0;
        let r2 = dot(barrel_uv, barrel_uv);
        barrel_uv *= 1.0 + 0.05 * r2;
        let dist_uv = barrel_uv * 0.5 + 0.5;

        // Vignette
        let vign = 1.0 - r2 * 0.4;

        // RGB aperture mask (3-pixel repeating: R,G,B)
        let mask_x = u32(in.pos.x) % 3u;
        var mask = vec3f(1.0);
        if mask_x == 0u { mask = vec3f(1.0, 0.7, 0.7); }
        else if mask_x == 1u { mask = vec3f(0.7, 1.0, 0.7); }
        else { mask = vec3f(0.7, 0.7, 1.0); }

        // Chromatic aberration at edges
        let ca = r2 * 0.003;
        let r_ch = textureSample(t_screen, s_screen, dist_uv + vec2f(ca, 0.0)).r;
        let b_ch = textureSample(t_screen, s_screen, dist_uv - vec2f(ca, 0.0)).b;
        col = vec3f(r_ch, col.g, b_ch);

        col *= mask * vign;

        // Clip to screen edges after barrel distortion
        if dist_uv.x < 0.0 || dist_uv.x > 1.0 || dist_uv.y < 0.0 || dist_uv.y > 1.0 {
            col = vec3f(0.0);
        }
    }

    return vec4f(col, 1.0);
}
```

### Task B3: Build the wgpu CRT render pipeline

**Files:**
- Create: `crates/frontend/src/crt_shader.rs`

**Step 1: Write the pipeline module:**
```rust
// crates/frontend/src/crt_shader.rs
//
// CRT post-process pipeline.
// Usage:
//   let crt = CrtPipeline::new(render_state);
//   // each frame:
//   crt.update_texture(render_state, &framebuffer.pixels, w, h);
//   crt.render(render_state, mode, dst_w, dst_h);
//   ui.image(crt.egui_texture_id(), size);

use bytemuck::{Pod, Zeroable};
use eframe::egui_wgpu::RenderState;
use egui::TextureId;
use wgpu::util::DeviceExt;

#[repr(C)]
#[derive(Copy, Clone, Pod, Zeroable)]
struct Uniforms {
    src_size: [f32; 2],
    dst_size: [f32; 2],
    mode: u32,
    _pad: u32,
}

#[derive(Clone, Copy, PartialEq, Default)]
pub enum CrtMode {
    #[default]
    Sharp = 0,
    Scanlines = 1,
    Crt = 2,
}

pub struct CrtPipeline {
    pipeline: wgpu::RenderPipeline,
    bind_group_layout: wgpu::BindGroupLayout,
    uniform_buf: wgpu::Buffer,
    sampler: wgpu::Sampler,

    // Source texture (emulator framebuffer)
    src_texture: wgpu::Texture,
    src_view: wgpu::TextureView,
    src_width: u32,
    src_height: u32,

    // Output texture (rendered into, then shown in egui)
    dst_texture: wgpu::Texture,
    dst_view: wgpu::TextureView,
    egui_tex_id: TextureId,

    bind_group: wgpu::BindGroup,
}

impl CrtPipeline {
    pub fn new(rs: &RenderState, src_w: u32, src_h: u32, dst_w: u32, dst_h: u32) -> Self {
        let device = &rs.device;
        let format = rs.target_format;

        let shader = device.create_shader_module(wgpu::ShaderModuleDescriptor {
            label: Some("crt"),
            source: wgpu::ShaderSource::Wgsl(include_str!("shaders/crt.wgsl").into()),
        });

        let bgl = device.create_bind_group_layout(&wgpu::BindGroupLayoutDescriptor {
            label: Some("crt_bgl"),
            entries: &[
                wgpu::BindGroupLayoutEntry { // texture
                    binding: 0, visibility: wgpu::ShaderStages::FRAGMENT,
                    ty: wgpu::BindingType::Texture {
                        sample_type: wgpu::TextureSampleType::Float { filterable: true },
                        view_dimension: wgpu::TextureViewDimension::D2,
                        multisampled: false,
                    },
                    count: None,
                },
                wgpu::BindGroupLayoutEntry { // sampler
                    binding: 1, visibility: wgpu::ShaderStages::FRAGMENT,
                    ty: wgpu::BindingType::Sampler(wgpu::SamplerBindingType::Filtering),
                    count: None,
                },
                wgpu::BindGroupLayoutEntry { // uniforms
                    binding: 2, visibility: wgpu::ShaderStages::FRAGMENT,
                    ty: wgpu::BindingType::Buffer {
                        ty: wgpu::BufferBindingType::Uniform,
                        has_dynamic_offset: false,
                        min_binding_size: None,
                    },
                    count: None,
                },
            ],
        });

        let pipeline_layout = device.create_pipeline_layout(&wgpu::PipelineLayoutDescriptor {
            label: Some("crt_pl"),
            bind_group_layouts: &[&bgl],
            push_constant_ranges: &[],
        });

        let pipeline = device.create_render_pipeline(&wgpu::RenderPipelineDescriptor {
            label: Some("crt_pipeline"),
            layout: Some(&pipeline_layout),
            vertex: wgpu::VertexState {
                module: &shader, entry_point: Some("vs_main"), buffers: &[],
                compilation_options: Default::default(),
            },
            fragment: Some(wgpu::FragmentState {
                module: &shader, entry_point: Some("fs_main"),
                compilation_options: Default::default(),
                targets: &[Some(wgpu::ColorTargetState {
                    format, blend: None,
                    write_mask: wgpu::ColorWrites::ALL,
                })],
            }),
            primitive: wgpu::PrimitiveState::default(),
            depth_stencil: None,
            multisample: wgpu::MultisampleState::default(),
            multiview: None,
            cache: None,
        });

        let sampler = device.create_sampler(&wgpu::SamplerDescriptor {
            mag_filter: wgpu::FilterMode::Nearest,
            min_filter: wgpu::FilterMode::Nearest,
            ..Default::default()
        });

        let uniform_buf = device.create_buffer_init(&wgpu::util::BufferInitDescriptor {
            label: Some("crt_uniforms"),
            contents: bytemuck::bytes_of(&Uniforms {
                src_size: [src_w as f32, src_h as f32],
                dst_size: [dst_w as f32, dst_h as f32],
                mode: 0, _pad: 0,
            }),
            usage: wgpu::BufferUsages::UNIFORM | wgpu::BufferUsages::COPY_DST,
        });

        let src_texture = Self::make_texture(device, src_w, src_h,
            wgpu::TextureUsages::TEXTURE_BINDING | wgpu::TextureUsages::COPY_DST);
        let src_view = src_texture.create_view(&Default::default());

        let dst_texture = Self::make_texture(device, dst_w, dst_h,
            wgpu::TextureUsages::TEXTURE_BINDING | wgpu::TextureUsages::RENDER_ATTACHMENT);
        let dst_view = dst_texture.create_view(&Default::default());

        // Register output texture with egui so we can display it as an image
        let egui_tex_id = rs.renderer.write().register_native_texture(
            device, &dst_view, wgpu::FilterMode::Linear,
        );

        let bind_group = Self::make_bind_group(device, &bgl, &src_view, &sampler, &uniform_buf);

        Self {
            pipeline, bind_group_layout: bgl, uniform_buf, sampler,
            src_texture, src_view, src_width: src_w, src_height: src_h,
            dst_texture, dst_view, egui_tex_id, bind_group,
        }
    }

    fn make_texture(device: &wgpu::Device, w: u32, h: u32, usage: wgpu::TextureUsages) -> wgpu::Texture {
        device.create_texture(&wgpu::TextureDescriptor {
            label: Some("crt_tex"),
            size: wgpu::Extent3d { width: w, height: h, depth_or_array_layers: 1 },
            mip_level_count: 1, sample_count: 1,
            dimension: wgpu::TextureDimension::D2,
            format: wgpu::TextureFormat::Rgba8UnormSrgb,
            usage,
            view_formats: &[],
        })
    }

    fn make_bind_group(
        device: &wgpu::Device,
        layout: &wgpu::BindGroupLayout,
        src_view: &wgpu::TextureView,
        sampler: &wgpu::Sampler,
        uniform_buf: &wgpu::Buffer,
    ) -> wgpu::BindGroup {
        device.create_bind_group(&wgpu::BindGroupDescriptor {
            label: Some("crt_bg"),
            layout,
            entries: &[
                wgpu::BindGroupEntry { binding: 0, resource: wgpu::BindingResource::TextureView(src_view) },
                wgpu::BindGroupEntry { binding: 1, resource: wgpu::BindingResource::Sampler(sampler) },
                wgpu::BindGroupEntry { binding: 2, resource: uniform_buf.as_entire_binding() },
            ],
        })
    }

    /// Upload new framebuffer pixels (RGBA8) to the source texture.
    pub fn update_pixels(&self, rs: &RenderState, pixels: &[u8], w: u32, h: u32) {
        rs.queue.write_texture(
            self.src_texture.as_image_copy(),
            pixels,
            wgpu::ImageDataLayout { offset: 0, bytes_per_row: Some(w * 4), rows_per_image: None },
            wgpu::Extent3d { width: w, height: h, depth_or_array_layers: 1 },
        );
    }

    /// Run the post-process pass. Call once per frame before displaying.
    pub fn render(&self, rs: &RenderState, mode: CrtMode, dst_w: u32, dst_h: u32) {
        rs.queue.write_buffer(&self.uniform_buf, 0, bytemuck::bytes_of(&Uniforms {
            src_size: [self.src_width as f32, self.src_height as f32],
            dst_size: [dst_w as f32, dst_h as f32],
            mode: mode as u32,
            _pad: 0,
        }));

        let mut encoder = rs.device.create_command_encoder(&Default::default());
        {
            let mut pass = encoder.begin_render_pass(&wgpu::RenderPassDescriptor {
                label: Some("crt_pass"),
                color_attachments: &[Some(wgpu::RenderPassColorAttachment {
                    view: &self.dst_view,
                    resolve_target: None,
                    ops: wgpu::Operations { load: wgpu::LoadOp::Clear(wgpu::Color::BLACK), store: wgpu::StoreOp::Store },
                })],
                depth_stencil_attachment: None,
                timestamp_writes: None,
                occlusion_query_set: None,
            });
            pass.set_pipeline(&self.pipeline);
            pass.set_bind_group(0, &self.bind_group, &[]);
            pass.draw(0..3, 0..1); // Full-screen triangle
        }
        rs.queue.submit(std::iter::once(encoder.finish()));
    }

    /// The egui TextureId to display the post-processed output.
    pub fn egui_texture_id(&self) -> TextureId {
        self.egui_tex_id
    }

    /// Recreate if source resolution changes (e.g. system switched).
    pub fn src_size(&self) -> (u32, u32) {
        (self.src_width, self.src_height)
    }
}
```

use egui::{Context, ViewportBuilder, ViewportId};
use emu_common::{Debuggable, CpuDebugState};

pub struct DebuggerWindow {
    pub open: bool,
    viewport_id: ViewportId,
    breakpoints: Vec<u16>,
    mem_addr: u16,
    follow_pc: bool,
    paused: bool,
}

impl DebuggerWindow {
    pub fn new() -> Self {
        Self {
            open: false,
            viewport_id: ViewportId::from_hash_of("debugger"),
            breakpoints: Vec::new(),
            mem_addr: 0,
            follow_pc: true,
            paused: false,
        }
    }

    pub fn is_paused(&self) -> bool { self.paused }

    /// Show the debugger as a deferred viewport (separate OS window).
    /// Call this every frame from the main app update.
    pub fn show(
        &mut self,
        ctx: &Context,
        system: &mut dyn SystemEmulatorWithDebug,
    ) {
        if !self.open { return; }

        let state = system.cpu_debug_state();
        if self.follow_pc { self.mem_addr = state.pc; }

        let vp_builder = ViewportBuilder::default()
            .with_title("Debugger")
            .with_inner_size([900.0, 700.0]);

        ctx.show_viewport_deferred(self.viewport_id, vp_builder, {
            // Clone needed data for the closure
            let breakpoints = self.breakpoints.clone();
            let mem_addr = self.mem_addr;
            let paused = self.paused;
            move |ctx, _class| {
                egui::CentralPanel::default().show(ctx, |ui| {
                    // Layout: left column (registers + disasm), right column (memory + system)
                    ui.columns(2, |cols| {
                        // Left: CPU state + disassembler
                        show_cpu_panel(&mut cols[0], &state);
                        show_disasm_panel(&mut cols[0], system, state.pc, &breakpoints, paused);
                        // Right: memory + system-specific
                        show_memory_panel(&mut cols[1], system, mem_addr);
                    });
                    // Bottom: system-specific panels
                    system.debugger_ui(ui);
                });
            }
        });
    }
}

fn show_cpu_panel(ui: &mut egui::Ui, s: &CpuDebugState) {
    ui.group(|ui| {
        ui.label(format!("PC:{:04X}  A:{:02X}  X:{:02X}  Y:{:02X}  SP:{:02X}",
            s.pc, s.a, s.x, s.y, s.sp));
        let f = s.flags;
        ui.label(format!("Flags: N:{} V:{} B:{} D:{} I:{} Z:{} C:{}",
            (f>>7)&1, (f>>6)&1, (f>>4)&1, (f>>3)&1, (f>>2)&1, (f>>1)&1, f&1));
        ui.label(format!("Cycles: {}", s.cycles));
    });
}

fn show_disasm_panel(
    ui: &mut egui::Ui,
    system: &dyn Debuggable,
    pc: u16,
    breakpoints: &[u16],
    _paused: bool,
) {
    ui.group(|ui| {
        egui::ScrollArea::vertical().show(ui, |ui| {
            let mut addr = pc.saturating_sub(10 * 3); // approx 10 instructions before PC
            for _ in 0..30 {
                let (text, next) = system.disassemble(addr);
                let is_pc = addr == pc;
                let is_bp = breakpoints.contains(&addr);
                let label = format!("{}{} {}",
                    if is_bp { "● " } else { "  " },
                    if is_pc { "▶" } else { " " },
                    text);
                if is_pc {
                    ui.colored_label(egui::Color32::YELLOW, &label);
                } else {
                    ui.monospace(&label);
                }
                addr = next;
            }
        });
    });
}

fn show_memory_panel(ui: &mut egui::Ui, system: &dyn Debuggable, base: u16) {
    ui.group(|ui| {
        for row in 0..16u16 {
            let addr = base.wrapping_add(row * 16);
            let bytes: Vec<u8> = (0..16u16).map(|i| system.debug_read(addr.wrapping_add(i))).collect();
            let hex: String = bytes.iter().map(|b| format!("{b:02X} ")).collect();
            let ascii: String = bytes.iter().map(|&b| if b.is_ascii_graphic() { b as char } else { '.' }).collect();
            ui.monospace(format!("{addr:04X}: {hex} {ascii}"));
        }
    });
}

// crates/frontend/src/save_state.rs
use std::path::{Path, PathBuf};
use std::hash::{Hash, Hasher};

const NUM_SLOTS: usize = 4;

pub fn save_dir(rom_path: &Path) -> PathBuf {
    // e.g. saves/c64/abc123def456/
    let hash = file_hash(rom_path);
    let system = rom_path.parent()
        .and_then(|p| p.file_name())
        .and_then(|n| n.to_str())
        .unwrap_or("unknown");
    PathBuf::from("saves").join(system).join(format!("{hash:016x}"))
}

fn file_hash(path: &Path) -> u64 {
    let mut h = std::collections::hash_map::DefaultHasher::new();
    path.hash(&mut h);
    std::hash::Hasher::finish(&h)
}

pub fn slot_path(rom_path: &Path, slot: usize) -> PathBuf {
    save_dir(rom_path).join(format!("slot{slot}.bin"))
}

pub fn save<T: serde::Serialize>(state: &T, path: &Path) -> Result<(), String> {
    std::fs::create_dir_all(path.parent().unwrap()).map_err(|e| e.to_string())?;
    let bytes = rmp_serde::to_vec(state).map_err(|e| e.to_string())?;
    std::fs::write(path, bytes).map_err(|e| e.to_string())
}

pub fn load<T: serde::de::DeserializeOwned>(path: &Path) -> Result<T, String> {
    let bytes = std::fs::read(path).map_err(|e| e.to_string())?;
    rmp_serde::from_slice(&bytes).map_err(|e| e.to_string())
}

pub fn slot_exists(rom_path: &Path, slot: usize) -> bool {
    slot_path(rom_path, slot).exists()
}
