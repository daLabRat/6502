**Goal:** Play a 7800 game with MARIA DMA sprite rendering; load and play a 2600 game via backward-compatibility mode.

**Architecture:** New `emu-atari7800` crate. The 7800 has a 6502C CPU (compatible with standard 6502), the MARIA chip (replaces GTIA, DMA-driven sprite engine), and TIA (audio only, reused logic — not shared from `emu-atari2600`, copied as needed). 2600 backward-compatibility mode: detect cartridge type, use TIA video + audio instead of MARIA.

**Tech Stack:** Rust, `emu-common`, `emu-cpu` (Cpu6502), `.a78` file format (16-byte header + ROM), TIA audio channel logic (copy from atari2600 crate).

## Context: What Exists

- No Atari 7800 crate exists.
- The Atari 7800 uses a 6502C (standard 6502 compatible — `bcd_enabled = true`).
- MARIA: sprite-oriented DMA engine. Reads object list from RAM, fetches sprite data, renders scanline by scanline.
- TIA: used for audio in 7800 mode. In 2600 compatibility mode, TIA also drives video.
- `.a78` ROM format: 128-byte header, then ROM data. Header byte $01 = format; $09-$0A = cart size; $11 = TV type; $12 = controller type.
- RIOT: same as Atari 2600 (6532 RAM + I/O + Timer) — copy/adapt from `emu-atari2600`.

```rust
/// TIA audio channels for Atari 7800.
/// Only AUDC/AUDF/AUDV are used; video is handled by MARIA.

const TIA_CLOCK_RATE: f64 = 3_579_545.0;
const AUDIO_CLOCK_DIVISOR: u16 = 114;

struct AudioChannel {
    audc: u8, audf: u8, audv: u8,
    freq_counter: u8,
    poly4: u8, poly5: u8, poly9: u16,
    div_counter: u8,
    output: bool,
}

impl AudioChannel {
    fn new() -> Self {
        Self { audc:0, audf:0, audv:0, freq_counter:0,
               poly4:0x0F, poly5:0x1F, poly9:0x1FF, div_counter:0, output:false }
    }

    fn step(&mut self) {
        if self.freq_counter == 0 {
            self.freq_counter = self.audf;
            self.clock_waveform();
        } else {
            self.freq_counter -= 1;
        }
    }

    fn clock_waveform(&mut self) {
        let p4 = ((self.poly4 >> 1) ^ self.poly4) & 1;
        self.poly4 = ((self.poly4 >> 1) | (p4 << 3)) & 0x0F;
        let p5 = ((self.poly5 >> 2) ^ self.poly5) & 1;
        self.poly5 = ((self.poly5 >> 1) | (p5 << 4)) & 0x1F;
        let p9 = (((self.poly9 >> 4) ^ self.poly9) & 1) as u16;
        self.poly9 = ((self.poly9 >> 1) | (p9 << 8)) & 0x1FF;

        match self.audc & 0x0F {
            0x00 | 0x0B => self.output = true,
            0x01 => self.output = self.poly4 & 1 != 0,
            0x04 | 0x05 => self.output = !self.output,
            0x06 | 0x0A => {
                self.div_counter = self.div_counter.wrapping_add(1);
                if self.div_counter >= 31 { self.div_counter = 0; self.output = !self.output; }
            }
            0x08 => self.output = self.poly9 & 1 != 0,
            0x09 => self.output = self.poly5 & 1 != 0,
            0x0C | 0x0D => {
                self.div_counter = self.div_counter.wrapping_add(1);
                if self.div_counter >= 6 { self.div_counter = 0; self.output = !self.output; }
            }
            _ => {}
        }
    }

    fn sample(&self) -> f32 {
        if self.output { self.audv as f32 / 15.0 } else { 0.0 }
    }
}

pub struct TiaAudio {
    ch: [AudioChannel; 2],
    audio_clock: u16,
    sample_rate: u32,
    sample_accum: f64,
    pub sample_buffer: Vec<f32>,
}

impl TiaAudio {
    pub fn new() -> Self {
        Self {
            ch: [AudioChannel::new(), AudioChannel::new()],
            audio_clock: 0,
            sample_rate: 44100,
            sample_accum: 0.0,
            sample_buffer: Vec::with_capacity(1024),
        }
    }

    pub fn set_sample_rate(&mut self, rate: u32) { self.sample_rate = rate; }

    pub fn write(&mut self, addr: u16, val: u8) {
        match addr & 0x3F {
            0x15 => self.ch[0].audc = val & 0x0F,
            0x16 => self.ch[1].audc = val & 0x0F,
            0x17 => self.ch[0].audf = val & 0x1F,
            0x18 => self.ch[1].audf = val & 0x1F,
            0x19 => self.ch[0].audv = val & 0x0F,
            0x1A => self.ch[1].audv = val & 0x0F,
            _ => {}
        }
    }

    pub fn step_clock(&mut self) {
        self.audio_clock += 1;
        if self.audio_clock >= AUDIO_CLOCK_DIVISOR {
            self.audio_clock = 0;
            self.ch[0].step();
            self.ch[1].step();
        }
        if self.sample_rate > 0 {
            self.sample_accum += self.sample_rate as f64;
            if self.sample_accum >= TIA_CLOCK_RATE {
                self.sample_accum -= TIA_CLOCK_RATE;
                self.sample_buffer.push((self.ch[0].sample() + self.ch[1].sample()) * 0.5);
            }
        }
    }

    pub fn drain_samples(&mut self, out: &mut [f32]) -> usize {
        let n = out.len().min(self.sample_buffer.len());
        out[..n].copy_from_slice(&self.sample_buffer[..n]);
        self.sample_buffer.drain(..n);
        n
    }
}
```

## Phase D: MARIA Sprite Engine

### Task D1: Implement MARIA

**Files:**
- Write: `crates/atari7800/src/maria.rs`

**Background:** MARIA reads an **object list** (a linked list of display list pointers) from RAM each frame. For each scanline, MARIA fetches the header and sprite data and renders directly to the framebuffer.

**MARIA registers ($D000-$D01F):**
```
$D000: BACKGRND — background color
$D001: P0C1 — player 0 color 1
$D002: P0C2 — player 0 color 2
$D003: P0C3 — player 0 color 3
$D004: P1C1 ... etc.
$D008: M0C1 (missile 0)
$D00C: M1C1 (missile 1)
$D010: Write: CHARBASE (address base for character data)
$D012: OFFSET (display list base address offset)
$D014: DPPH/DPPL — display list pointer (high/low)
$D018: CTRL — control register
$D01C: MSTAT — status register (read)
$D01D: MSTAT — write clears VBL
```

**MARIA object list structure:**
Each scanline's sprite list is: `[header][sprite 0 width][sprite 0 data ptr lo][sprite 0 data ptr hi][x pos][...]`

The actual MARIA object list is complex. For a functional first implementation:
1. MARIA reads `DPPH:DPPL` to find the display list
2. For each entry, it reads a 5-byte header: `[DLL ptr hi, lo, offset, palette, mode]`
3. Within each line, it reads sprite descriptors: `[ptr hi, lo, xpos, palette, width]`

This is simplified to get games running; full DL accuracy requires hardware documentation.

```rust
use emu_common::FrameBuffer;

pub const SCREEN_WIDTH: u32  = 320;
pub const SCREEN_HEIGHT: u32 = 240;

/// NTSC 7800 color palette (256 colors → RGB24).
/// Identical to Atari 7800 NTSC palette.
fn atari7800_color(c: u8) -> u32 {
    // Use same hue/luminance encoding as Atari 8-bit
    let hue = (c >> 4) as u32;
    let lum = ((c & 0x0E) as u32) * 17;
    match hue {
        0  => (lum << 16) | (lum << 8) | lum,   // grey
        1  => (lum << 16) | ((lum/2) << 8),      // gold
        2  => (lum << 16) | ((lum/3) << 8),      // orange
        3  => lum << 16,                           // red
        4  => (lum << 16) | ((lum/4) << 8) | lum, // pink-purple
        5  => ((lum/2) << 16) | lum,              // purple
        6  => lum,                                  // dark blue
        7  => ((lum/4) << 8) | lum,               // blue
        8  => ((lum/2) << 8) | lum,               // medium blue
        9  => (lum << 8) | lum,                   // cyan
        10 => (lum << 8) | (lum/2),               // blue-green
        11 => lum << 8,                             // green
        12 => ((lum/4) << 16) | (lum << 8),       // yellow-green
        13 => ((lum/2) << 16) | (lum << 8),       // yellow-green 2
        14 => (lum << 16) | (lum << 8),           // yellow
        _  => (lum << 16) | ((lum/2) << 8),       // orange-yellow
    }
}

pub struct Maria {
    pub regs: [u8; 32],
    pub backgrnd: u8,
    pub dpph: u8,
    pub dppl: u8,
    pub ctrl: u8,
    pub charbase: u8,

    scanline: u16,
    pub frame_ready: bool,
    pub framebuffer: FrameBuffer,
}

impl Maria {
    pub fn new() -> Self {
        Self {
            regs: [0; 32],
            backgrnd: 0,
            dpph: 0, dppl: 0,
            ctrl: 0,
            charbase: 0,
            scanline: 0,
            frame_ready: false,
            framebuffer: FrameBuffer::new(SCREEN_WIDTH, SCREEN_HEIGHT),
        }
    }

    pub fn read(&self, addr: u16) -> u8 {
        match addr & 0x1F {
            0x1C => if self.scanline >= SCREEN_HEIGHT as u16 { 0x80 } else { 0x00 }, // MSTAT VBL
            _ => self.regs[(addr & 0x1F) as usize],
        }
    }

    pub fn write(&mut self, addr: u16, val: u8) {
        let r = (addr & 0x1F) as usize;
        self.regs[r] = val;
        match r {
            0x00 => self.backgrnd = val,
            0x10 => self.charbase = val,
            0x12 => {}, // OFFSET
            0x14 => self.dpph = val,
            0x15 => self.dppl = val,
            0x18 => self.ctrl = val,
            0x1D => {} // MSTAT write — clear VBL
            _ => {}
        }
    }

    /// Step one scanline. Reads from `memory` to render sprites.
    /// Returns true when a new frame is ready.
    pub fn step_scanline(&mut self, memory: &[u8]) -> bool {
        let vis_start = 16u16;
        let vis_end   = vis_start + SCREEN_HEIGHT as u16;

        if self.scanline == 0 {
            self.frame_ready = false;
            // Fill background
            let bg = atari7800_color(self.backgrnd);
            for y in 0..SCREEN_HEIGHT {
                for x in 0..SCREEN_WIDTH {
                    self.framebuffer.set_pixel_rgb(x, y, bg);
                }
            }
        }

        if self.scanline >= vis_start && self.scanline < vis_end {
            let vis_y = (self.scanline - vis_start) as u32;
            self.render_scanline(memory, vis_y);
        }

        self.scanline += 1;
        if self.scanline >= 262 {
            self.scanline = 0;
            self.frame_ready = true;
            return true;
        }
        false
    }

    /// Render one scanline using MARIA's display list.
    fn render_scanline(&mut self, memory: &[u8], y: u32) {
        // MARIA display list pointer
        let dll_base = u16::from_le_bytes([self.dppl, self.dpph]);
        if dll_base == 0 { return; }

        // Find DL entry for this scanline (simplified: walk DL up to y entries)
        // Real MARIA uses a zone-based approach — this is a functional approximation
        let dll_entry = dll_base.wrapping_add(y as u16 * 3);
        let dl_hi  = memory.get(dll_entry as usize).copied().unwrap_or(0);
        let dl_lo  = memory.get(dll_entry as usize + 1).copied().unwrap_or(0);
        let _flags = memory.get(dll_entry as usize + 2).copied().unwrap_or(0);

        let dl_ptr = u16::from_le_bytes([dl_lo, dl_hi]);
        if dl_ptr == 0 { return; }

        // Walk the display list for this scanline
        let mut ptr = dl_ptr;
        loop {
            let width = memory.get(ptr as usize).copied().unwrap_or(0);
            if width == 0 { break; } // End of line marker
            ptr += 1;

            let data_lo = memory.get(ptr as usize).copied().unwrap_or(0);
            ptr += 1;
            let data_hi = memory.get(ptr as usize).copied().unwrap_or(0);
            ptr += 1;
            let xpos = memory.get(ptr as usize).copied().unwrap_or(0) as u32;
            ptr += 1;
            let palette = memory.get(ptr as usize).copied().unwrap_or(0);
            ptr += 1;

            let data_ptr = u16::from_le_bytes([data_lo, data_hi]);

            // Render sprite pixels
            for px in 0..(width as u32) {
                let byte_idx = px / 4;
                let bit_pair = (px % 4) as u8;
                let data_byte = memory.get(data_ptr as usize + byte_idx as usize).copied().unwrap_or(0);
                let color_idx = (data_byte >> (6 - bit_pair * 2)) & 0x03;
                if color_idx == 0 { continue; } // transparent

                let color_reg = (palette & 0x1C) | color_idx;
                let color = self.regs[color_reg as usize];
                let sx = xpos + px;
                if sx < SCREEN_WIDTH {
                    self.framebuffer.set_pixel_rgb(sx, y, atari7800_color(color));
                }
            }
        }
    }
}

/// Atari 7800 cartridge (.a78 format or raw binary).
pub struct Cartridge {
    pub rom: Vec<u8>,
    pub load_addr: u16,
    pub is_2600: bool,
}

impl Cartridge {
    /// Load from .a78 file (128-byte header + ROM).
    pub fn from_a78(data: &[u8]) -> Result<Self, String> {
        if data.len() < 128 { return Err("Too short for .a78".into()); }
        // Magic at offset 1: "ATARI7800"
        let magic = &data[1..10];
        let is_7800 = magic == b"ATARI7800";
        let is_2600 = magic == b"ATARI2600";

        if !is_7800 && !is_2600 {
            // Try raw binary
            return Ok(Self::from_raw(data));
        }

        let cart_size = u32::from_be_bytes([data[0x09], data[0x0A], data[0x0B], data[0x0C]]);
        let rom_data = data[128..].to_vec();
        let _ = cart_size;

        // 7800 ROMs map at $4000-$FFFF (32KB) or $8000-$FFFF (16KB)
        let load_addr = if rom_data.len() <= 16384 { 0xC000 } else { 0x4000 };
        Ok(Self { rom: rom_data, load_addr, is_2600 })
    }

    pub fn from_raw(data: &[u8]) -> Self {
        let load_addr = if data.len() <= 16384 { 0xC000 } else { 0x4000 };
        Self { rom: data.to_vec(), load_addr, is_2600: false }
    }

    pub fn read(&self, addr: u16) -> u8 {
        let offset = addr.wrapping_sub(self.load_addr) as usize;
        self.rom.get(offset).copied().unwrap_or(0xFF)
    }

    pub fn in_range(&self, addr: u16) -> bool {
        addr >= self.load_addr
    }
}

use emu_common::Bus;
use crate::maria::Maria;
use crate::tia_audio::TiaAudio;
use crate::riot::Riot;
use crate::cartridge::Cartridge;

/// Atari 7800 memory map:
/// $0000-$17FF  RAM (6KB mirrored — $0000-$03FF internal, rest expansion)
/// $1800-$27FF  RAM (mirrored)
/// $2800-$2FFF  MARIA registers ($D000 in some docs, depends on mode)
/// $4000-$FFFF  Cartridge ROM
/// $C000-$CFFF  BIOS ROM (7800)
/// $D000-$D01F  MARIA
/// $D400-$D7FF  TIA registers (audio)
/// $D800-$DBFF  RIOT registers
/// $DC00-$DFFF  RIOT RAM
pub struct Atari7800Bus {
    pub ram:  [u8; 0x1800], // 6KB
    pub bios: Vec<u8>,      // 7800 BIOS (4KB at $C000)
    pub cart: Cartridge,
    pub maria: Maria,
    pub tia:   TiaAudio,
    pub riot:  Riot,
    cycles_per_scanline: u32,
    cycle_count: u32,
    pub nmi_pending: bool,
}

impl Atari7800Bus {
    pub fn new(bios: Vec<u8>, cart: Cartridge) -> Self {
        Self {
            ram: [0; 0x1800],
            bios,
            cart,
            maria: Maria::new(),
            tia: TiaAudio::new(),
            riot: Riot::new(),
            cycles_per_scanline: 113,
            cycle_count: 0,
            nmi_pending: false,
        }
    }
}

impl Bus for Atari7800Bus {
    fn read(&mut self, addr: u16) -> u8 {
        self.cart.check_bank_switch_if_any(addr);
        match addr {
            0x0000..=0x17FF => self.ram[(addr & 0x17FF) as usize],
            0x1800..=0x27FF => self.ram[(addr & 0x07FF) as usize],
            0xC000..=0xCFFF => self.bios.get((addr - 0xC000) as usize).copied().unwrap_or(0xFF),
            0xD000..=0xD01F => self.maria.read(addr - 0xD000),
            0xD400..=0xD7FF => 0, // TIA read (audio-only, simplified)
            0xD800..=0xDBFF => self.riot.read(addr),
            0xDC00..=0xDFFF => self.riot.ram[(addr & 0x7F) as usize],
            _ if self.cart.in_range(addr) => self.cart.read(addr),
            _ => 0xFF,
        }
    }

    fn write(&mut self, addr: u16, val: u8) {
        match addr {
            0x0000..=0x17FF => self.ram[(addr & 0x17FF) as usize] = val,
            0x1800..=0x27FF => self.ram[(addr & 0x07FF) as usize] = val,
            0xD000..=0xD01F => self.maria.write(addr - 0xD000, val),
            0xD400..=0xD7FF => self.tia.write(addr - 0xD400, val),
            0xD800..=0xDBFF => self.riot.write(addr, val),
            0xDC00..=0xDFFF => self.riot.ram[(addr & 0x7F) as usize] = val,
            _ => {}
        }
    }

    fn peek(&self, addr: u16) -> u8 {
        match addr {
            0x0000..=0x17FF => self.ram[(addr & 0x17FF) as usize],
            0xC000..=0xCFFF => self.bios.get((addr - 0xC000) as usize).copied().unwrap_or(0xFF),
            _ if self.cart.in_range(addr) => self.cart.read(addr),
            _ => 0xFF,
        }
    }

    fn tick(&mut self, cycles: u8) {
        for _ in 0..cycles {
            self.tia.step_clock(); self.tia.step_clock(); self.tia.step_clock();
            self.riot.step();
            self.cycle_count += 1;
            if self.cycle_count >= self.cycles_per_scanline {
                self.cycle_count = 0;
                let ram_view: Vec<u8> = self.ram.to_vec();
                let frame_done = self.maria.step_scanline(&ram_view);
                if frame_done && self.maria.ctrl & 0x80 != 0 {
                    self.nmi_pending = true;
                }
            }
        }
    }

    fn poll_nmi(&mut self) -> bool {
        let n = self.nmi_pending;
        self.nmi_pending = false;
        n
    }

    fn poll_irq(&mut self) -> bool {
        self.riot.irq_pending()
    }
}
```

impl SystemEmulator for Atari7800 {
    fn step_frame(&mut self) -> usize {
        self.cpu.bus.maria.frame_ready = false;
        while !self.cpu.bus.maria.frame_ready {
            self.cpu.step();
        }
        0
    }

    fn framebuffer(&self) -> &FrameBuffer { &self.cpu.bus.maria.framebuffer }

    fn audio_samples(&mut self, out: &mut [AudioSample]) -> usize {
        self.cpu.bus.tia.drain_samples(out)
    }

    fn handle_input(&mut self, event: InputEvent) {
        // Atari 7800 joystick — RIOT SWCHA like 2600
        let mask = if event.port == 0 {
            match event.button {
                Button::Up    => Some(0x10u8),
                Button::Down  => Some(0x20),
                Button::Left  => Some(0x40),
                Button::Right => Some(0x80),
                _ => None,
            }
        } else {
            match event.button {
                Button::Up    => Some(0x01),
                Button::Down  => Some(0x02),
                Button::Left  => Some(0x04),
                Button::Right => Some(0x08),
                _ => None,
            }
        };
        if let Some(m) = mask {
            if event.pressed { self.cpu.bus.riot.swcha &= !m; }
            else             { self.cpu.bus.riot.swcha |=  m; }
        }
    }

    fn reset(&mut self) { self.cpu.reset(); }
    fn set_sample_rate(&mut self, rate: u32) { self.cpu.bus.tia.set_sample_rate(rate); }

    fn display_width(&self)  -> u32 { maria::SCREEN_WIDTH }
    fn display_height(&self) -> u32 { maria::SCREEN_HEIGHT }
    fn target_fps(&self)     -> f64 { 59.94 }
    fn system_name(&self)    -> &str { "Atari 7800" }
    fn save_state_system_id(&self) -> &str { "Atari7800" }

    fn cpu_state(&self) -> CpuDebugState {
        CpuDebugState { pc: self.cpu.pc, sp: self.cpu.sp, a: self.cpu.a,
            x: self.cpu.x, y: self.cpu.y, flags: self.cpu.p.bits(), cycles: self.cpu.total_cycles }
    }
    fn peek_memory(&self, addr: u16) -> u8 { self.cpu.bus.peek(addr) }
    fn disassemble(&self, addr: u16) -> (String, u16) {
        emu_cpu::disassemble_6502(|a| self.cpu.bus.peek(a), addr)
    }
    fn step_instruction(&mut self) { self.cpu.step(); }

    fn system_debug_panels(&self) -> Vec<DebugSection> {
        let maria = &self.cpu.bus.maria;
        let riot  = &self.cpu.bus.riot;
        let maria_sec = DebugSection::new("MARIA")
            .row("Scanline", format!("{}", maria.scanline))
            .row("DL Ptr",   format!("${:02X}{:02X}", maria.dpph, maria.dppl))
            .row("CTRL",     format!("${:02X}", maria.ctrl))
            .row("BG Color", format!("${:02X}", maria.backgrnd));
        let riot_sec = DebugSection::new("RIOT")
            .row("Timer",  format!("${:02X}", riot.timer_value()))
            .row("SWCHA",  format!("{:08b}", riot.swcha))
            .row("SWCHB",  format!("{:08b}", riot.swchb));
        vec![maria_sec, riot_sec]
    }
}
