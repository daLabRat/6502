# Slice 5E: Commodore PET — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Boot the Commodore PET 2001 to a BASIC prompt and run BASIC programs, with correct 40×25 character-mapped display and tape loading support.

**Architecture:** New `emu-pet` crate. The PET is the simplest new system: 6502 CPU, 6545 CRTC for display, character ROM, two 6520 PIAs for keyboard and IEEE-488, one 6522 VIA for tape interface, and 32KB RAM. No sound hardware (beyond cassette relay). The CRTC generates MA/RA for screen RAM fetch; the display is rendered as a grid of character cells from screen RAM + character ROM.

**Tech Stack:** Rust, `emu-common`, `emu-cpu` (Cpu6502). ROMs: BASIC (16KB), KERNAL (a.k.a. "editor" ROM, 4KB), character ROM (2KB). Display: 40×25 text, each character 8×8 pixels → 320×200 pixels.

---

## Context: What Exists

- No PET crate exists.
- The PET uses a 6502 at ~1 MHz (various models: 1, 2, or 4 MHz).
- 6545 CRTC: very similar to 6845 used in BBC Micro. Same register layout, generates MA[13:0] and RA[4:0].
- 6520 PIA: two 8-bit ports (A and B), two control registers. Simpler than 6522.
- 6522 VIA: same as in C64 and BBC Micro.
- Screen RAM: $8000-$83E7 (1000 bytes for 40×25 screen). Characters are PETSCII codes.
- Character ROM: $E800-$EFFF (not CPU-accessible in normal mode; accessed via CRTC).

## Phase B: 6520 PIA

### Task B1: Implement 6520 PIA

**Files:**
- Write: `crates/pet/src/pia.rs`

**Background:** The 6520 PIA (Peripheral Interface Adapter) has:
- Port A: 8-bit bidirectional I/O (PA0-PA7)
- Port B: 8-bit bidirectional I/O (PB0-PB7)
- Control registers CA1/CA2 and CB1/CB2 edge-sensitive inputs
- Unlike 6522: no timers, no shift register

**PIA register map (offsets within the chip's 4-register window):**
```
RS0=0, RS1=0  ($00 or $04 depending on chip select): ORA or DDRA (selected by CRA bit 2)
RS0=1, RS1=0  ($01 or $05): CRA — Control Register A
RS0=0, RS1=1  ($02 or $06): ORB or DDRB (selected by CRB bit 2)
RS0=1, RS1=1  ($03 or $07): CRB — Control Register B
```

**Step 1: Write `crates/pet/src/pia.rs`:**
```rust
/// 6520 PIA (Peripheral Interface Adapter).
pub struct Pia {
    pub ora:  u8,  // Output Register A
    pub orb:  u8,  // Output Register B
    pub ddra: u8,  // Data Direction Register A (1=output)
    pub ddrb: u8,  // Data Direction Register B
    pub cra:  u8,  // Control Register A
    pub crb:  u8,  // Control Register B
    pub ira:  u8,  // Input Register A (external)
    pub irb:  u8,  // Input Register B (external)
}

impl Pia {
    pub fn new() -> Self {
        Self { ora:0, orb:0, ddra:0, ddrb:0, cra:0, crb:0, ira:0xFF, irb:0xFF }
    }

    pub fn read(&mut self, addr: u16) -> u8 {
        match addr & 0x03 {
            0x00 => {
                if self.cra & 0x04 != 0 {
                    // ORA selected: output bits from ORA, input bits from IRA
                    self.cra &= !0xC0; // Clear IRQ flags on Port A read
                    (self.ora & self.ddra) | (self.ira & !self.ddra)
                } else {
                    self.ddra
                }
            }
            0x01 => self.cra,
            0x02 => {
                if self.crb & 0x04 != 0 {
                    self.crb &= !0xC0;
                    (self.orb & self.ddrb) | (self.irb & !self.ddrb)
                } else {
                    self.ddrb
                }
            }
            0x03 => self.crb,
            _ => 0,
        }
    }

    pub fn write(&mut self, addr: u16, val: u8) {
        match addr & 0x03 {
            0x00 => {
                if self.cra & 0x04 != 0 { self.ora = val; } else { self.ddra = val; }
            }
            0x01 => self.cra = val & 0x3F, // Bits 6-7 are read-only IRQ flags
            0x02 => {
                if self.crb & 0x04 != 0 { self.orb = val; } else { self.ddrb = val; }
            }
            0x03 => self.crb = val & 0x3F,
            _ => {}
        }
    }

    /// Returns effective Port A output (driven bits).
    pub fn port_a_out(&self) -> u8 { (self.ora & self.ddra) | (!self.ddra) }
    pub fn port_b_out(&self) -> u8 { (self.orb & self.ddrb) | (!self.ddrb) }
}
```

## Phase E: Video Renderer

### Task E1: Character display renderer

**Files:**
- Write: `crates/pet/src/video.rs`

**Background:** The PET display is 40 characters × 25 rows. Each character is 8×8 pixels. The character ROM provides 256 glyphs × 8 bytes/glyph. Screen RAM is at $8000-$83E7 (CPU address space).

For CRTC address space: The PET maps screen RAM at CRTC address $0000 (MA=0 → $8000 CPU, via external logic). Character ROM is accessed separately by the video hardware at MA offset (not by CPU directly).

**Display output: 40×8 = 320 pixels wide, 25×8 = 200 pixels high.**

**PETSCII to character ROM mapping:**
- Codes $00-$1F: uppercase glyphs 0-31
- Codes $20-$3F: punctuation + digits
- Codes $40-$5F: uppercase A-Z + symbols
- Codes $60-$7F: lowercase or graphics
- Codes $80-$FF: reverse video versions of $00-$7F

**Step 1: Write `crates/pet/src/video.rs`:**
```rust
use emu_common::FrameBuffer;

pub const SCREEN_WIDTH:  u32 = 320; // 40 cols × 8 pixels
pub const SCREEN_HEIGHT: u32 = 200; // 25 rows × 8 pixels

pub struct Video {
    pub char_rom: Vec<u8>,    // 2KB character ROM (256 chars × 8 bytes)
    pub framebuffer: FrameBuffer,
    pub frame_ready: bool,
    render_col: u32,
    render_row: u32,
}

impl Video {
    pub fn new(char_rom: Vec<u8>) -> Self {
        Self {
            char_rom,
            framebuffer: FrameBuffer::new(SCREEN_WIDTH, SCREEN_HEIGHT),
            frame_ready: false,
            render_col: 0,
            render_row: 0,
        }
    }

    /// Render one character cell from screen RAM + char ROM.
    /// Called by the bus each time CRTC advances one character position.
    /// `screen_byte`: PETSCII code from screen RAM
    /// `ra`: row address from CRTC (0-7 for 8-pixel-high chars)
    /// `display`: whether CRTC says this cell is in the visible area
    /// `frame_end`: CRTC frame complete signal
    pub fn render_cell(&mut self, screen_byte: u8, ra: u8, display: bool, frame_end: bool) {
        if frame_end {
            self.render_col = 0;
            self.render_row = 0;
            self.frame_ready = true;
            return;
        }

        if !display { return; }

        let x = self.render_col * 8;
        let y = self.render_row * 8 + ra as u32;
        if x >= SCREEN_WIDTH || y >= SCREEN_HEIGHT {
            self.render_col += 1;
            if self.render_col >= 40 {
                self.render_col = 0;
                if ra == 7 { self.render_row += 1; }
            }
            return;
        }

        // Determine reverse video
        let reverse = screen_byte & 0x80 != 0;
        let char_code = (screen_byte & 0x7F) as usize;

        // Character ROM: glyph_data[char_code * 8 + ra]
        let glyph_byte = self.char_rom.get(char_code * 8 + ra as usize).copied().unwrap_or(0);

        // Render 8 pixels
        for bit in 0..8u32 {
            let pixel_set = glyph_byte & (0x80 >> bit) != 0;
            let lit = if reverse { !pixel_set } else { pixel_set };
            let color = if lit { 0x33FF33u32 } else { 0x000000u32 }; // Green phosphor
            self.framebuffer.set_pixel_rgb(x + bit, y, color);
        }

        self.render_col += 1;
        if self.render_col >= 40 {
            self.render_col = 0;
            if ra == 7 { self.render_row += 1; }
        }
    }
}
```

## Phase F: Bus + SystemEmulator + Frontend

### Task F1: PET Bus

**Files:**
- Write: `crates/pet/src/bus.rs`

**PET memory map (32KB model):**
```
$0000-$7FFF  RAM (32KB)
$8000-$83E7  Screen RAM (1000 bytes = 40×25)
$8800-$8FFF  I/O space
  $8800-$880F  PIA1 ($E810 in some docs, mapped here in 32KB)
  $8810-$881F  PIA2
  $8820-$882F  VIA
  $8830        CRTC address register
  $8831        CRTC data register
$C000-$DFFF  BASIC ROM (8KB or 16KB)
$E000-$E3FF  BASIC ROM continued
$E800-$EFFF  KERNAL ROM (editor + OS)
$F000-$FFFF  KERNAL ROM
```

Note: PET ROM layout varied by model. We target the PET 2001 (40-column), 32KB:
- BASIC 4.0: $B000-$DFFF (12KB)
- KERNAL/editor: $E000-$FFFF (8KB)

**Step 1: Write `crates/pet/src/bus.rs`:**
```rust
use emu_common::Bus;
use crate::crtc::Crtc;
use crate::pia::Pia;
use crate::via::Via;
use crate::video::Video;

pub struct PetBus {
    pub ram:        [u8; 0x8000], // 32KB
    pub screen_ram: [u8; 0x400],  // 1KB ($8000-$83FF, only $3E8 used)
    pub basic_rom:  Vec<u8>,      // 12KB or 16KB BASIC
    pub kernal_rom: Vec<u8>,      // 4KB or 8KB KERNAL
    pub crtc:       Crtc,
    pub pia1:       Pia,          // Keyboard + IEEE-488
    pub pia2:       Pia,          // User + cassette
    pub via:        Via,          // Tape motor + misc
    pub video:      Video,

    crtc_div:  u32,  // CRTC at 1 MHz vs CPU at 1 MHz (1:1 ratio)
}

impl PetBus {
    pub fn new(basic_rom: Vec<u8>, kernal_rom: Vec<u8>, char_rom: Vec<u8>) -> Self {
        Self {
            ram:        [0; 0x8000],
            screen_ram: [0; 0x400],
            basic_rom,
            kernal_rom,
            crtc: Crtc::new(),
            pia1: Pia::new(),
            pia2: Pia::new(),
            via:  Via::new(),
            video: Video::new(char_rom),
            crtc_div: 0,
        }
    }

    /// Read byte from CRTC screen RAM address space (MA-relative).
    fn crtc_read_screen(&self, ma: u16) -> u8 {
        // CRTC MA 0 → screen RAM $8000 (CPU) → screen_ram[0]
        self.screen_ram.get(ma as usize).copied().unwrap_or(0x20) // Default = space
    }
}

impl Bus for PetBus {
    fn read(&mut self, addr: u16) -> u8 {
        match addr {
            0x0000..=0x7FFF => self.ram[addr as usize],
            0x8000..=0x83FF => self.screen_ram[(addr - 0x8000) as usize],
            0x8800..=0x880F => self.pia1.read(addr - 0x8800),
            0x8810..=0x881F => self.pia2.read(addr - 0x8810),
            0x8820..=0x882F => self.via.read(addr - 0x8820),
            0x8830 => 0, // CRTC address register (write-only)
            0x8831 => self.crtc.read_data(),
            0xB000..=0xDFFF => self.basic_rom.get((addr - 0xB000) as usize).copied().unwrap_or(0xFF),
            0xE000..=0xFFFF => self.kernal_rom.get((addr - 0xE000) as usize).copied().unwrap_or(0xFF),
            _ => 0xFF,
        }
    }

    fn write(&mut self, addr: u16, val: u8) {
        match addr {
            0x0000..=0x7FFF => self.ram[addr as usize] = val,
            0x8000..=0x83FF => self.screen_ram[(addr - 0x8000) as usize] = val,
            0x8800..=0x880F => self.pia1.write(addr - 0x8800, val),
            0x8810..=0x881F => self.pia2.write(addr - 0x8810, val),
            0x8820..=0x882F => self.via.write(addr - 0x8820, val),
            0x8830 => self.crtc.write_addr(val),
            0x8831 => self.crtc.write_data(val),
            _ => {}
        }
    }

    fn peek(&self, addr: u16) -> u8 {
        match addr {
            0x0000..=0x7FFF => self.ram[addr as usize],
            0x8000..=0x83FF => self.screen_ram[(addr - 0x8000) as usize],
            0xB000..=0xDFFF => self.basic_rom.get((addr - 0xB000) as usize).copied().unwrap_or(0xFF),
            0xE000..=0xFFFF => self.kernal_rom.get((addr - 0xE000) as usize).copied().unwrap_or(0xFF),
            _ => 0xFF,
        }
    }

    fn tick(&mut self, cycles: u8) {
        for _ in 0..cycles {
            self.via.step();

            // CRTC steps at same rate as CPU (1 MHz)
            self.crtc.step();
            let ma = self.crtc.ma;
            let ra = self.crtc.ra;
            let screen_byte = self.crtc_read_screen(ma);
            self.video.render_cell(screen_byte, ra, self.crtc.display, self.crtc.frame_end);
            self.crtc.frame_end = false;
        }
    }

    fn poll_nmi(&mut self) -> bool { false }
    fn poll_irq(&mut self) -> bool { self.via.irq_pending() }
}
```

**Step 2: Write `crates/pet/src/lib.rs`:**
```rust
pub mod bus;
pub mod crtc;
pub mod pia;
pub mod via;
pub mod video;

use emu_common::{AudioSample, Button, CpuDebugState, DebugSection, FrameBuffer, InputEvent, SystemEmulator};
use emu_cpu::Cpu6502;
use bus::PetBus;

pub struct Pet {
    cpu: Cpu6502<PetBus>,
}

impl Pet {
    pub fn with_roms(basic: Vec<u8>, kernal: Vec<u8>, chargen: Vec<u8>) -> Self {
        let bus = PetBus::new(basic, kernal, chargen);
        let mut cpu = Cpu6502::new(bus);
        cpu.bcd_enabled = true;
        cpu.reset();
        Self { cpu }
    }

    /// Inject a BASIC program as if typed.
    /// PRG format: 2-byte load address + tokenized BASIC.
    pub fn load_prg(&mut self, data: &[u8]) -> Result<(), String> {
        if data.len() < 3 { return Err("PRG too short".into()); }
        let addr = u16::from_le_bytes([data[0], data[1]]);
        for (i, &b) in data[2..].iter().enumerate() {
            let a = addr.wrapping_add(i as u16) as usize;
            if a < self.cpu.bus.ram.len() {
                self.cpu.bus.ram[a] = b;
            }
        }
        Ok(())
    }
}

impl SystemEmulator for Pet {
    fn step_frame(&mut self) -> usize {
        self.cpu.bus.video.frame_ready = false;
        while !self.cpu.bus.video.frame_ready {
            self.cpu.step();
        }
        0
    }

    fn framebuffer(&self) -> &FrameBuffer { &self.cpu.bus.video.framebuffer }

    fn audio_samples(&mut self, _out: &mut [AudioSample]) -> usize {
        0 // PET has no sound hardware
    }

    fn handle_input(&mut self, event: InputEvent) {
        // PET keyboard: scanned through PIA1
        // Port B selects keyboard row, Port A reads columns
        // Minimal implementation: no keyboard input
        let _ = event;
    }

    fn reset(&mut self) { self.cpu.reset(); }
    fn set_sample_rate(&mut self, _rate: u32) {}

    fn display_width(&self)  -> u32 { video::SCREEN_WIDTH }
    fn display_height(&self) -> u32 { video::SCREEN_HEIGHT }
    fn target_fps(&self)     -> f64 { 60.0 }
    fn system_name(&self)    -> &str { "Commodore PET" }
    fn save_state_system_id(&self) -> &str { "CommodorePET" }

    fn cpu_state(&self) -> CpuDebugState {
        CpuDebugState { pc: self.cpu.pc, sp: self.cpu.sp, a: self.cpu.a,
            x: self.cpu.x, y: self.cpu.y, flags: self.cpu.p.bits(), cycles: self.cpu.total_cycles }
    }
    fn peek_memory(&self, addr: u16) -> u8 { self.cpu.bus.peek(addr) }
    fn disassemble(&self, addr: u16) -> (String, u16) {
        emu_cpu::disassemble_6502(|a| self.cpu.bus.peek(a), addr)
    }
    fn step_instruction(&mut self) { self.cpu.step(); }

    fn system_debug_panels(&self) -> Vec<DebugSection> {
        let crtc = &self.cpu.bus.crtc;
        let pia1 = &self.cpu.bus.pia1;
        let pia2 = &self.cpu.bus.pia2;
        let via  = &self.cpu.bus.via;

        let crtc_sec = DebugSection::new("6545 CRTC")
            .row("MA/RA",   format!("${:04X} / {}", crtc.ma, crtc.ra))
            .row("H/V cnt", format!("{} / {}", crtc.h_count, crtc.v_count))
            .row("Start",   format!("${:04X}", crtc.start_address()))
            .row("Display", format!("{}", crtc.display));

        let pia1_sec = DebugSection::new("PIA1 (Kbd)")
            .row("Port A",  format!("out=${:02X} ddr=${:02X}", pia1.ora, pia1.ddra))
            .row("Port B",  format!("out=${:02X} ddr=${:02X}", pia1.orb, pia1.ddrb));

        let pia2_sec = DebugSection::new("PIA2 (Tape)")
            .row("Port A",  format!("out=${:02X} ddr=${:02X}", pia2.ora, pia2.ddra))
            .row("Port B",  format!("out=${:02X} ddr=${:02X}", pia2.orb, pia2.ddrb));

        let via_sec = DebugSection::new("VIA (Tape motor)")
            .row("T1",  format!("${:04X} (latch ${:04X})", via.t1_counter, via.t1_latch))
            .row("IRQ", format!("ifr=${:02X} ier=${:02X}", via.ifr, via.ier));

        vec![crtc_sec, pia1_sec, pia2_sec, via_sec]
    }
}
```
