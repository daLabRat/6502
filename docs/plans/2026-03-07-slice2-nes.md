## Phase A: APU Accuracy

### Task A1: Fix frame counter half-cycle timing

**Files:**
- Modify: `crates/nes/src/apu/mod.rs`

**Background:** The NES APU frame counter fires on CPU half-cycles. The integer thresholds should be offset by treating the counter as incrementing at the *start* of each cycle. The standard correct counts (NESDev wiki):

- 4-step mode: fire at CPU cycles 3728, 7456, 11185, 14914 (end-of-cycle, so counter == value after increment)
- IRQ flag set at cycle 14914 and again at 0 (wrap)
- 5-step mode: fire at 3728, 7456, 11185, 18640

The current code fires at 3729, 7457, 11186, 14915 — one cycle late.

**Step 1: Update the frame counter match arms** in `Apu::step()`:
```rust
// 4-step sequence (mode 0)
match self.frame_counter {
    3728 => self.clock_quarter_frame(),
    7456 => { self.clock_quarter_frame(); self.clock_half_frame(); }
    11185 => self.clock_quarter_frame(),
    14914 => {
        if !self.frame_irq_inhibit { self.frame_irq_pending = true; }
    }
    14915 => {
        self.clock_quarter_frame();
        self.clock_half_frame();
        if !self.frame_irq_inhibit { self.frame_irq_pending = true; }
        self.frame_counter = 0;
    }
    _ => {}
}

// 5-step sequence (mode 1)
match self.frame_counter {
    3728  => self.clock_quarter_frame(),
    7456  => { self.clock_quarter_frame(); self.clock_half_frame(); }
    11185 => self.clock_quarter_frame(),
    18640 => { self.clock_quarter_frame(); self.clock_half_frame(); }
    18641 => { self.frame_counter = 0; }
    _ => {}
}
```

### Task A2: Fix sweep unit mute condition

**Files:**
- Modify: `crates/nes/src/apu/pulse.rs`

**Background:** The pulse channel should be silenced when the *target period* (after applying sweep shift) would exceed $7FF, not the current period. Also the current period must be ≥ 8 to produce output regardless of sweep.

**Step 1: Compute target period in `output()`** and mute based on it:
```rust
pub fn output(&self) -> u8 {
    // Silence if length counter expired
    if self.length_counter == 0 { return 0; }
    // Silence if timer period too low (< 8 = ultrasonic)
    if self.timer_period < 8 { return 0; }
    // Silence if sweep target would overflow (> $7FF)
    let change = self.timer_period >> self.sweep_shift;
    let target = if self.sweep_negate {
        self.timer_period.saturating_sub(change)
            .saturating_sub(if self.is_pulse1 { 1 } else { 0 })
    } else {
        self.timer_period.saturating_add(change)
    };
    if target > 0x7FF { return 0; }
    // Silence if duty cycle is low
    if DUTY_TABLE[self.duty as usize][self.duty_pos as usize] == 0 { return 0; }
    // Volume
    if self.constant_volume { self.envelope_period } else { self.envelope_decay }
}
```

**Note:** Keep the existing `clock_sweep()` — it already correctly only updates `timer_period` when target ≤ $7FF. This change only affects the mute check in `output()`.

### Task A3: Add DMC DMA CPU stall

**Files:**
- Modify: `crates/nes/src/bus.rs`

**Background:** When the DMC DMA reads a byte, it steals 1–4 CPU cycles depending on alignment. For simplicity, always stall 4 cycles (the maximum, which is what most emulators use for correctness without cycle-perfect CPU alignment tracking).

**Step 1: Track accumulated DMC stall in `tick()`:**

The DMC DMA is currently serviced inside the `for _ in 0..total_cpu_cycles` loop. Each DMA read should add 4 stall cycles to the APU/PPU but NOT re-run CPU instructions.

Change the tick loop to accumulate stall cycles and run extra APU/PPU ticks:

```rust
fn tick(&mut self, cycles: u8) {
    let dma_stall = if self.oam_dma_pending {
        self.oam_dma_pending = false;
        let page = (self.oam_dma_page as u16) << 8;
        let mut data = [0u8; 256];
        for i in 0..256u16 {
            data[i as usize] = self.read(page | i);
        }
        self.ppu.write_oam_dma(&data);
        513u32
    } else {
        0
    };

    let total_cpu = cycles as u32 + dma_stall;
    let mut extra_apu = 0u32;

    // PPU: 3 cycles per CPU cycle
    for _ in 0..total_cpu * 3 {
        self.ppu.step(self.cartridge.mapper.as_mut());
    }

    // APU at CPU rate; service DMC DMA inline
    for _ in 0..total_cpu {
        self.apu.step();
        self.cartridge.mapper.cpu_tick();

        if let Some(addr) = self.apu.dmc.dma_request.take() {
            let byte = match addr {
                0x0000..=0x1FFF => self.ram[(addr & 0x07FF) as usize],
                0x8000..=0xFFFF => self.cartridge.mapper.cpu_read(addr),
                _ => 0,
            };
            self.apu.dmc.receive_dma_byte(byte);
            extra_apu += 4; // DMC DMA stall: 4 extra APU/PPU cycles
        }
    }

    // Run the extra stall cycles through APU + PPU
    for _ in 0..extra_apu {
        self.apu.step();
    }
    for _ in 0..extra_apu * 3 {
        self.ppu.step(self.cartridge.mapper.as_mut());
    }

    if self.ppu.nmi_pending {
        self.ppu_nmi_pending = true;
        self.ppu.nmi_pending = false;
    }
}
```

### Task B1: Implement VRC6 mapper

**Files:**
- Create: `crates/nes/src/cartridge/mapper/vrc6.rs`
- Modify: `crates/nes/src/cartridge/mapper/mod.rs`

**Background:** VRC6 exists in two pin configurations (mapper 24 = VRC6a, mapper 26 = VRC6b). They differ only in which address lines carry A0/A1. VRC6 features:
- PRG: 16KB fixed bank at $C000–$FFFF + two switchable 8KB banks at $8000/$A000
- CHR: 8 switchable 1KB banks
- IRQ: scanline counter with prescaler
- Expansion audio: 2 pulse channels + 1 sawtooth

**Step 1: Write VRC6 mapper** (`crates/nes/src/cartridge/mapper/vrc6.rs`):
```rust
pub struct Vrc6 {
    prg_rom: Vec<u8>,
    chr_rom: Vec<u8>,       // may be CHR RAM if chr_rom.len() == 0
    chr_ram: Vec<u8>,
    mirroring: super::super::Mirroring,

    // PRG banking
    prg_bank_8k_a: u8,   // $8000-$9FFF
    prg_bank_8k_b: u8,   // $A000-$BFFF
    // $C000-$FFFF: fixed to last 16KB

    // CHR banking (8x 1KB)
    chr_banks: [u8; 8],

    // IRQ
    irq_enabled: bool,
    irq_pending: bool,
    irq_latch: u8,
    irq_counter: u8,
    irq_prescaler: u8,
    irq_mode: bool,     // false=scanline, true=CPU cycle

    // A0/A1 swap for VRC6b (mapper 26)
    swap_ab: bool,
}

impl Vrc6 {
    pub fn new(prg_rom: Vec<u8>, chr_rom: Vec<u8>, mirroring: super::super::Mirroring, swap_ab: bool) -> Self {
        let chr_ram = if chr_rom.is_empty() { vec![0u8; 8192] } else { vec![] };
        Self {
            prg_rom, chr_rom, chr_ram, mirroring,
            prg_bank_8k_a: 0, prg_bank_8k_b: 0,
            chr_banks: [0; 8],
            irq_enabled: false, irq_pending: false,
            irq_latch: 0, irq_counter: 0, irq_prescaler: 0, irq_mode: false,
            swap_ab,
        }
    }

    fn normalize_addr(&self, addr: u16) -> u16 {
        if self.swap_ab {
            // VRC6b: swap A0 and A1
            let a0 = (addr >> 0) & 1;
            let a1 = (addr >> 1) & 1;
            (addr & !0x03) | (a0 << 1) | (a1 << 0)
        } else {
            addr
        }
    }

    fn prg_addr(bank_8k: u8, addr: u16, prg_len: usize) -> usize {
        let bank_size = 0x2000usize;
        let offset = (addr & 0x1FFF) as usize;
        ((bank_8k as usize * bank_size) % prg_len) + offset
    }
}

impl super::Mapper for Vrc6 {
    fn cpu_read(&self, addr: u16) -> u8 {
        match addr {
            0x6000..=0x7FFF => 0xFF, // No WRAM on VRC6 (most games)
            0x8000..=0x9FFF => {
                let i = Self::prg_addr(self.prg_bank_8k_a, addr, self.prg_rom.len());
                self.prg_rom[i]
            }
            0xA000..=0xBFFF => {
                let i = Self::prg_addr(self.prg_bank_8k_b, addr, self.prg_rom.len());
                self.prg_rom[i]
            }
            0xC000..=0xFFFF => {
                // Fixed: last 16KB
                let len = self.prg_rom.len();
                let offset = (addr - 0xC000) as usize;
                self.prg_rom[(len - 0x4000 + offset) % len]
            }
            _ => 0xFF,
        }
    }

    fn cpu_write(&mut self, addr: u16, val: u8) {
        let addr = self.normalize_addr(addr);
        match addr {
            // PRG bank selects ($8000, $C000)
            0x8000 => self.prg_bank_8k_a = val & 0x0F,
            0xC000 => self.prg_bank_8k_b = val & 0x1F,
            // CHR banks ($D000-$E003)
            0xD000 => self.chr_banks[0] = val,
            0xD001 => self.chr_banks[1] = val,
            0xD002 => self.chr_banks[2] = val,
            0xD003 => self.chr_banks[3] = val,
            0xE000 => self.chr_banks[4] = val,
            0xE001 => self.chr_banks[5] = val,
            0xE002 => self.chr_banks[6] = val,
            0xE003 => self.chr_banks[7] = val,
            // Mirroring ($B003)
            0xB003 => {
                self.mirroring = match (val >> 2) & 0x03 {
                    0 => super::super::Mirroring::Vertical,
                    1 => super::super::Mirroring::Horizontal,
                    2 => super::super::Mirroring::SingleScreenLow,
                    _ => super::super::Mirroring::SingleScreenHigh,
                };
            }
            // IRQ ($F000-$F002)
            0xF000 => self.irq_latch = val,
            0xF001 => {
                self.irq_mode = val & 0x04 != 0;
                self.irq_enabled = val & 0x02 != 0;
                if self.irq_enabled {
                    self.irq_counter = self.irq_latch;
                    self.irq_prescaler = 341;
                }
                self.irq_pending = false;
            }
            0xF002 => {
                self.irq_enabled = val & 0x01 != 0;
                self.irq_pending = false;
            }
            // Expansion audio registers ($9000-$9002, $A000-$A002, $B000-$B002)
            // Stubbed for now — add VRC6 audio in future
            _ => {}
        }
    }

    fn ppu_read(&mut self, addr: u16) -> u8 {
        if addr >= 0x2000 { return 0; }
        let bank = (addr / 0x0400) as usize;
        let offset = (addr & 0x03FF) as usize;
        let chr_bank = self.chr_banks[bank] as usize;
        if !self.chr_rom.is_empty() {
            self.chr_rom[(chr_bank * 0x400 + offset) % self.chr_rom.len()]
        } else {
            self.chr_ram[(chr_bank * 0x400 + offset) % self.chr_ram.len()]
        }
    }

    fn ppu_write(&mut self, addr: u16, val: u8) {
        if addr < 0x2000 && self.chr_rom.is_empty() {
            let bank = (addr / 0x0400) as usize;
            let offset = (addr & 0x03FF) as usize;
            let chr_bank = self.chr_banks[bank] as usize;
            let i = (chr_bank * 0x400 + offset) % self.chr_ram.len();
            self.chr_ram[i] = val;
        }
    }

    fn mirroring(&self) -> super::super::Mirroring { self.mirroring }

    fn cpu_tick(&mut self) {
        if !self.irq_enabled { return; }
        if self.irq_mode {
            // CPU cycle mode
            if self.irq_counter == 0xFF {
                self.irq_counter = self.irq_latch;
                self.irq_pending = true;
            } else {
                self.irq_counter += 1;
            }
        } else {
            // Scanline mode: prescaler counts to 341 (CPU cycles per scanline)
            self.irq_prescaler = self.irq_prescaler.saturating_sub(3);
            if self.irq_prescaler == 0 {
                self.irq_prescaler = 341;
                if self.irq_counter == 0xFF {
                    self.irq_counter = self.irq_latch;
                    self.irq_pending = true;
                } else {
                    self.irq_counter += 1;
                }
            }
        }
    }

    fn irq_pending(&self) -> bool { self.irq_pending }
    fn irq_clear(&mut self) { self.irq_pending = false; }
    fn mapper_state(&self) -> Vec<u8> { vec![] } // TODO: save state
    fn restore_mapper_state(&mut self, _data: &[u8]) {}
}
```

## Phase C: Mapper 19 — Namco 163

**Notable games:** Pac-Man (NES), Dig Dug, Galaxian, many Namco NES titles

### Task C1: Implement Namco 163 mapper

**Files:**
- Create: `crates/nes/src/cartridge/mapper/namco163.rs`
- Modify: `crates/nes/src/cartridge/mapper/mod.rs`

**Background:** Namco 163 features:
- PRG: four 8KB switchable banks at $8000/$A000/$C000, fixed last 8KB at $E000
- CHR: eight 1KB banks (some games use CHR RAM)
- Nametable: up to 2KB internal VRAM, nametable banks can point to CHR ROM
- IRQ: CPU cycle counter (counts up to $7FFF, fires at that value)
- Expansion audio: up to 8 channels of wavetable synthesis (complex — stub for now)

```rust
pub struct Namco163 {
    prg_rom: Vec<u8>,
    chr_rom: Vec<u8>,
    chr_ram: Vec<u8>,
    mirroring: super::super::Mirroring,

    // PRG banks (8KB each)
    prg_banks: [u8; 3],   // $8000, $A000, $C000 — $E000 is fixed

    // CHR banks (1KB each) — 8 CHR + 4 NT banks (indices 8-11)
    chr_banks: [u16; 12],

    // IRQ
    irq_enabled: bool,
    irq_pending: bool,
    irq_counter: u16,

    // Internal VRAM for nametable (2KB)
    vram: Vec<u8>,

    // Audio address register (stubbed)
    audio_addr: u8,
}

impl Namco163 {
    pub fn new(prg_rom: Vec<u8>, chr_rom: Vec<u8>, mirroring: super::super::Mirroring) -> Self {
        let chr_ram = if chr_rom.is_empty() { vec![0u8; 8192] } else { vec![] };
        Self {
            prg_rom, chr_rom, chr_ram, mirroring,
            prg_banks: [0; 3],
            chr_banks: [0; 12],
            irq_enabled: false, irq_pending: false, irq_counter: 0,
            vram: vec![0; 2048],
            audio_addr: 0,
        }
    }

    fn read_chr(&self, bank: u16, offset: usize) -> u8 {
        if !self.chr_rom.is_empty() {
            let addr = (bank as usize * 0x400 + offset) % self.chr_rom.len();
            self.chr_rom[addr]
        } else {
            let addr = (bank as usize * 0x400 + offset) % self.chr_ram.len().max(1);
            self.chr_ram[addr]
        }
    }
}

impl super::Mapper for Namco163 {
    fn cpu_read(&self, addr: u16) -> u8 {
        match addr {
            0x8000..=0x9FFF => {
                let b = self.prg_banks[0] as usize;
                self.prg_rom[(b * 0x2000 + (addr - 0x8000) as usize) % self.prg_rom.len()]
            }
            0xA000..=0xBFFF => {
                let b = self.prg_banks[1] as usize;
                self.prg_rom[(b * 0x2000 + (addr - 0xA000) as usize) % self.prg_rom.len()]
            }
            0xC000..=0xDFFF => {
                let b = self.prg_banks[2] as usize;
                self.prg_rom[(b * 0x2000 + (addr - 0xC000) as usize) % self.prg_rom.len()]
            }
            0xE000..=0xFFFF => {
                // Fixed: last 8KB
                let len = self.prg_rom.len();
                self.prg_rom[(len.saturating_sub(0x2000) + (addr - 0xE000) as usize) % len]
            }
            _ => 0xFF,
        }
    }

    fn cpu_write(&mut self, addr: u16, val: u8) {
        match addr {
            // CHR banks low ($8000-$DFFF even addresses)
            0x8000..=0xDFFF if addr & 1 == 0 => {
                let idx = ((addr - 0x8000) / 0x800) as usize;
                if idx < 8 {
                    self.chr_banks[idx] = (self.chr_banks[idx] & 0xFF00) | val as u16;
                }
            }
            0x8001..=0xDFFF if addr & 1 == 1 => {
                let idx = ((addr - 0x8001) / 0x800) as usize;
                if idx < 8 {
                    self.chr_banks[idx] = (self.chr_banks[idx] & 0x00FF) | ((val as u16 & 0x01) << 8);
                }
            }
            // PRG banks
            0xE000 => {
                self.prg_banks[0] = val & 0x3F;
                self.irq_counter = (self.irq_counter & 0x7F00) | val as u16;
            }
            0xE001 => self.irq_counter = (self.irq_counter & 0x00FF) | ((val as u16 & 0x7F) << 8),
            0xE002 => {
                self.prg_banks[1] = val & 0x3F;
                self.irq_enabled = val & 0x80 != 0;
                if !self.irq_enabled { self.irq_pending = false; }
            }
            0xE003 => self.prg_banks[2] = val & 0x3F,
            // Audio (stubbed)
            0xF800 => self.audio_addr = val & 0x7F,
            0x4800 => { /* audio data write, stubbed */ }
            _ => {}
        }
    }

    fn ppu_read(&mut self, addr: u16) -> u8 {
        if addr >= 0x2000 { return 0; }
        let bank_idx = (addr / 0x400) as usize;
        let offset = (addr & 0x03FF) as usize;
        self.read_chr(self.chr_banks[bank_idx], offset)
    }

    fn ppu_write(&mut self, addr: u16, val: u8) {
        if addr < 0x2000 && self.chr_rom.is_empty() {
            let bank_idx = (addr / 0x400) as usize;
            let offset = (addr & 0x03FF) as usize;
            let i = (self.chr_banks[bank_idx] as usize * 0x400 + offset) % self.chr_ram.len().max(1);
            self.chr_ram[i] = val;
        }
    }

    fn mirroring(&self) -> super::super::Mirroring { self.mirroring }

    fn cpu_tick(&mut self) {
        if !self.irq_enabled { return; }
        let next = self.irq_counter.wrapping_add(1) & 0x7FFF;
        if next == 0x7FFF {
            self.irq_pending = true;
        }
        self.irq_counter = next;
    }

    fn irq_pending(&self) -> bool { self.irq_pending }
    fn irq_clear(&mut self) { self.irq_pending = false; }
    fn mapper_state(&self) -> Vec<u8> { vec![] }
    fn restore_mapper_state(&mut self, _data: &[u8]) {}
}
```

    egui::CollapsingHeader::new("PPU — Pattern Tables").show(ui, |ui| {
        // Render two 16x16 tile grids (128x128 pixels each) using PPU CHR data.
        // Each tile = 8x8 pixels, 2bpp, palette index 0 used for color mapping.
        // Display as egui ColorImage updated each frame.
        // Palette selector: radio buttons 0-3
        ui.label("(pattern table viewer — 128x128 pixel grid per half)");
        // TODO: implement pixel rendering from CHR data
    });

    egui::CollapsingHeader::new("PPU — OAM Sprites").show(ui, |ui| {
        let ppu = &nes.cpu.bus.ppu;
        for i in 0..64usize {
            let base = i * 4;
            let y   = ppu.oam[base];
            let tile = ppu.oam[base + 1];
            let attr = ppu.oam[base + 2];
            let x   = ppu.oam[base + 3];
            if y < 0xEF {
                ui.monospace(format!(
                    "#{:02} ({:3},{:3}) tile:{:02X} pal:{} flip:{}{}",
                    i, x, y, tile, attr & 3,
                    if attr & 0x40 != 0 { "H" } else { "" },
                    if attr & 0x80 != 0 { "V" } else { "" },
                ));
            }
        }
    });

    egui::CollapsingHeader::new("APU").show(ui, |ui| {
        let apu = &nes.cpu.bus.apu;
        ui.label(format!("Pulse1:  len={:3} period={:4} vol={}",
            apu.pulse1.length_counter, apu.pulse1.timer_period, apu.pulse1.output()));
        ui.label(format!("Pulse2:  len={:3} period={:4} vol={}",
            apu.pulse2.length_counter, apu.pulse2.timer_period, apu.pulse2.output()));
        ui.label(format!("Triangle: len={:3} period={:4}",
            apu.triangle.length_counter, apu.triangle.timer_period));
        ui.label(format!("Noise:   len={:3} period={:4}",
            apu.noise.length_counter, apu.noise.timer_period));
        ui.label(format!("DMC:     bytes_remaining={} level={} irq={}",
            apu.dmc.bytes_remaining(), apu.dmc.output(), apu.dmc.irq_pending));
        ui.label(format!("Frame IRQ: {}", apu.frame_irq_pending));
    });

    egui::CollapsingHeader::new("Mapper").show(ui, |ui| {
        let cart = &nes.cpu.bus.cartridge;
        ui.label(format!("Mapper: {} | Mirroring: {:?}", cart.mapper_id, cart.mapper.mirroring()));
        ui.label(format!("IRQ pending: {}", cart.mapper.irq_pending()));
    });
}

**Step 1: Add `ppu_peek_chr(addr: u16) -> u8`** to PPU — reads CHR via the mapper without side effects:
```rust
pub fn ppu_peek_chr(&self, mapper: &dyn Mapper, addr: u16) -> u8 {
    // Use mapper's ppu_read (note: mmc2/mmc4 have latch side effects, but
    // peek is best-effort for the debugger)
    mapper.ppu_read(addr) // this is &mut — need a peek variant on Mapper trait
}
```

Add `fn ppu_peek(&self, addr: u16) -> u8 { 0 }` default to `Mapper` trait, override in mappers that can implement it without side effects.

**Step 2: Render 128×128 pattern table images** in `nes_debugger_ui()`:
```rust
// Build a 128×128 RGBA8 image from CHR data
let mut pixels = vec![0u8; 128 * 128 * 4];
for tile_y in 0..16u16 {
    for tile_x in 0..16u16 {
        let tile_num = tile_y * 16 + tile_x;
        for row in 0..8u16 {
            let addr = half * 0x1000 + tile_num * 16 + row;
            let lo = mapper.ppu_peek(addr);
            let hi = mapper.ppu_peek(addr + 8);
            for col in 0..8u16 {
                let bit = 7 - col;
                let px = ((lo >> bit) & 1) | (((hi >> bit) & 1) << 1);
                let gray = [0, 85, 170, 255][px as usize];
                let px_x = (tile_x * 8 + col) as usize;
                let px_y = (tile_y * 8 + row) as usize;
                let i = (px_y * 128 + px_x) * 4;
                pixels[i] = gray; pixels[i+1] = gray; pixels[i+2] = gray; pixels[i+3] = 255;
            }
        }
    }
}
// Load as egui texture and display
let img = egui::ColorImage::from_rgba_unmultiplied([128, 128], &pixels);
let tex = ui.ctx().load_texture("chr_left", img, egui::TextureOptions::NEAREST);
ui.image(egui::load::SizedTexture::new(tex.id(), egui::vec2(128.0, 128.0)));
```

- Disk I/O IRQ (transfer-complete interrupt)
