## Phase A: Disk II Write Support

### Task A1: Capture written nibbles in the track buffer

**Files:**
- Modify: `crates/apple2/src/disk_ii.rs`

**Background:** When the firmware writes a disk sector, it:
1. Sets write mode (Q7H: switch `0xF`)
2. Loads each nibble byte via `STA $C0ED` (Q6H write: switch `0xD`, passes byte in A)
3. The controller shifts the byte to the disk head

We need to capture those nibble bytes into a mutable copy of the track data.

**Step 1: Add a dirty write buffer to `DiskII`:**
```rust
pub struct DiskII {
    // ... existing fields ...
    /// Mutable nibble track data (written over nibble_data on write).
    /// Initialised as a clone of nibble_data when a disk is loaded.
    write_data: Vec<[u8; TRACK_NIBBLE_SIZE]>,
    /// Which tracks have been written since last save.
    dirty_tracks: Vec<bool>,
    /// Write latch: byte staged by STA $C0ED, written on next step tick.
    write_latch: u8,
    write_latch_ready: bool,
}
```

Initialize in `new()`:
```rust

dirty_tracks: Vec::new(),
write_latch: 0,
write_latch_ready: false,
```

In `load_dsk()`, after building `nibble_data`, clone it:
```rust

```

**Step 2: Capture nibble in `io_write` Q6H handler:**

**Step 3: Also capture on `io_read` Q6H (some firmware reads Q6H to time writes):**
```rust

```

**Step 4: Flush the write latch in `step()` when in write mode:**
```rust
pub fn step(&mut self, cycles: u8) {
    if !self.motor_on || !self.disk_loaded { return; }

    self.cycle_accumulator += cycles as u32;
    while self.cycle_accumulator >= 32 {
        self.cycle_accumulator -= 32;
        self.byte_position = (self.byte_position + 1) % TRACK_NIBBLE_SIZE;

        if self.write_mode {
            // Write the staged latch byte to the track buffer
            if self.write_latch_ready {
                let track = self.current_track as usize;
                if track < self.write_data.len() {
                    self.write_data[track][self.byte_position] = self.write_latch;
                    self.dirty_tracks[track] = true;
                }
                self.write_latch_ready = false;
            }
        } else {
            // Read: supply nibble from whichever buffer is current
            // Use write_data (reflects any writes done this session)
            let track = self.current_track as usize;
            if let Some(track_data) = self.write_data.get(track) {
                self.data_latch = track_data[self.byte_position];
            }
        }
    }
}
```

### Task A2: Track denibblizer (write_data → DSK sectors)

**Files:**
- Modify: `crates/apple2/src/disk_ii.rs`

**Background:** To recover 256-byte sectors from the written nibble stream, we need to scan for:
1. Address field: `D5 AA 96` prologue → volume/track/sector/checksum → `DE AA EB` epilogue
2. Data field: `D5 AA AD` prologue → 343 nibble bytes (6-and-2 encoded) → `DE AA` epilogue

The `WRITE_TABLE` already exists for encoding. Build its inverse for decoding.

**Step 1: Build the read (decode) table** as a `const` or `static`:
```rust
/// Inverse of WRITE_TABLE: maps disk nibble byte → 6-bit value (0xFF = invalid).
static READ_TABLE: [u8; 256] = {
    let mut t = [0xFFu8; 256];
    let mut i = 0usize;
    while i < 64 {
        t[WRITE_TABLE[i] as usize] = i as u8;
        i += 1;
    }
    t
};
```

**Note:** `const` fn loops are stable in Rust — use a `const fn build_read_table()` if `while` in const context isn't available, or compute at startup in `new()`.

**Step 2: Add 4-and-4 decode helper** (for address field bytes):
```rust
fn decode_4and4(a: u8, b: u8) -> u8 {
    ((a & 0x55) << 1) | (b & 0x55)
}
```

**Step 3: Add `denibblize_track(nibbles: &[u8]) -> [[u8; 256]; 16]`:**
```rust
fn denibblize_track(nibbles: &[u8]) -> [[u8; 256]; 16] {
    let mut sectors = [[0u8; 256]; 16];
    let len = nibbles.len();
    let mut pos = 0;

    // Scan for up to 16 sectors
    let mut found = 0;
    while found < 16 {
        // Find D5 AA 96 (address field prologue)
        let start = find_sequence(nibbles, pos, &[0xD5, 0xAA, 0x96]);
        if start.is_none() { break; }
        pos = start.unwrap() + 3;

        // Read address field: volume, track, sector, checksum (each 4-and-4 encoded)
        if pos + 7 >= len { break; }
        let _volume  = decode_4and4(nibbles[pos],   nibbles[pos+1]);
        let _track   = decode_4and4(nibbles[pos+2], nibbles[pos+3]);
        let sector   = decode_4and4(nibbles[pos+4], nibbles[pos+5]);
        // skip checksum + epilogue DE AA EB
        pos += 8;

        if sector >= 16 { continue; }

        // Find D5 AA AD (data field prologue)
        let dstart = find_sequence(nibbles, pos, &[0xD5, 0xAA, 0xAD]);
        if dstart.is_none() { break; }
        pos = dstart.unwrap() + 3;

        // 343 nibble bytes = 86 2-bit pairs + 256 6-bit values + 1 checksum
        if pos + 343 > len { break; }
        let data_nibbles = &nibbles[pos..pos + 343];
        pos += 343 + 3; // skip DE AA checksum

        sectors[sector as usize] = decode_6and2(data_nibbles);
        found += 1;
    }

    sectors
}

/// Find byte sequence in a circular buffer. Returns index of first byte of sequence.
fn find_sequence(data: &[u8], start: usize, seq: &[u8]) -> Option<usize> {
    let len = data.len();
    for i in 0..len {
        let pos = (start + i) % len;
        if (0..seq.len()).all(|j| data[(pos + j) % len] == seq[j]) {
            return Some(pos);
        }
    }
    None
}
```

**Step 4: Add `get_modified_dsk(&self, original_order: SectorOrder) -> Option<Vec<u8>>`:**
```rust
pub fn get_modified_dsk(&self) -> Option<Vec<u8>> {
    if !self.dirty_tracks.iter().any(|&d| d) {
        return None; // Nothing written
    }

    let table = &DOS33_PHYSICAL_TO_LOGICAL; // TODO: detect and preserve original order
    let mut dsk = vec![0u8; 143360];

    for track in 0..NUM_TRACKS {
        let nibbles = &self.write_data[track];
        let sectors = denibblize_track(nibbles);
        for physical in 0..16 {
            let logical = table[physical];
            let offset = track * 4096 + logical * 256;
            dsk[offset..offset + 256].copy_from_slice(&sectors[physical]);
        }
    }

    Some(dsk)
}

/// Clear dirty flags (call after save).
pub fn clear_dirty(&mut self) {
    for d in &mut self.dirty_tracks { *d = false; }
}

pub fn is_dirty(&self) -> bool {
    self.dirty_tracks.iter().any(|&d| d)
}
```

## Phase B: 80-Column Text Rendering

### Task B1: Render 80-column text mode

**Files:**
- Modify: `crates/apple2/src/video/text.rs`
- Modify: `crates/apple2/src/video/mod.rs`

**Background:** In 80-column mode, the screen shows 80 characters per line × 24 rows. The IIe 80-column card stores even columns in main text page ($0400–$07FF) and odd columns in aux text page (same addresses in `aux_ram`). Character cells are 7 pixels wide × 8 pixels tall → 560 pixels wide (already our framebuffer width).

**Step 1: Add `render_text_80col()` to `text.rs`:**
```rust
pub fn render_text_80col(
    fb: &mut FrameBuffer,
    memory: &Memory,
    switches: &SoftSwitches,
    flash_on: bool,
) {
    render_text_80col_lines(fb, memory, switches, 0, 24, flash_on);
}

pub fn render_text_80col_lines(
    fb: &mut FrameBuffer,
    memory: &Memory,
    switches: &SoftSwitches,
    first_row: usize,
    last_row: usize,
    flash_on: bool,
) {
    for row in first_row..last_row {
        for col in 0..80usize {
            // Even columns from main text page, odd columns from aux text page
            let text_addr = text_row_addr(row) + (col / 2) as u16;
            let ch = if col % 2 == 0 {
                memory.read_main_text(text_addr)
            } else {
                memory.read_aux_text(text_addr)
            };
            // Render 7x8 character cell at pixel x=(col*7), y=(row*8)
            render_char_cell_7wide(fb, ch, col * 7, row * 8, switches.altcharset, flash_on);
        }
    }
}
```

**Step 2: Add `render_char_cell_7wide()`** — same as existing `render_char_cell()` but uses 7-pixel width instead of 14 (the 80-col character ROM cells are 7px vs the 40-col 14px):
```rust
fn render_char_cell_7wide(
    fb: &mut FrameBuffer,
    ch: u8,
    px: usize, py: usize,
    altcharset: bool,
    flash_on: bool,
) {
    // Apple IIe character ROM: 256 chars × 8 rows × 7 bits/row
    // In 80-col mode, each character is 7 pixels wide (not doubled)
    let (display_ch, inverse, flash) = char_mode(ch, altcharset);
    let invert = inverse || (flash && !flash_on);
    for row in 0..8usize {
        let bits = char_rom_row(display_ch, row); // 7 bits
        for col in 0..7usize {
            let bit = (bits >> (6 - col)) & 1;
            let lit = if invert { bit == 0 } else { bit != 0 };
            let color = if lit { WHITE } else { BLACK };
            fb.set_pixel((px + col) as u32, (py + row) as u32, color);
        }
    }
}
```

### Task C2: Render DHGR

**Files:**
- Modify: `crates/apple2/src/video/hires.rs`

**Background:** DHGR produces 560 pixels per line by interleaving 4 bits from aux hi-res and 4 bits from main hi-res per byte pair. Each pair of bytes (aux byte, main byte) contributes 7 pixels: bits 0-6 from aux, bits 0-6 from main (7 bits each, bit 7 is palette control in regular hi-res but ignored in DHGR).

**Step 1: Add `render_dhgr()` to `hires.rs`:**
```rust
pub fn render_dhgr(
    fb: &mut FrameBuffer,
    memory: &Memory,
    switches: &SoftSwitches,
    lines: usize,
) {
    let page_base: u16 = if switches.page2 { 0x4000 } else { 0x2000 };

    for y in 0..lines {
        let row_addr = hires_row_addr(y as u16, page_base);
        let mut px = 0usize;

        for byte_col in 0..40usize {
            let addr = row_addr + byte_col as u16;
            let aux_byte  = memory.read_aux_hires(addr);
            let main_byte = memory.read_hires(addr);

            // 7 bits from aux, then 7 bits from main = 14 pixels per column pair
            for bit in 0..7usize {
                let color = if (aux_byte >> bit) & 1 != 0 { WHITE } else { BLACK };
                fb.set_pixel(px as u32, y as u32, color);
                px += 1;
            }
            for bit in 0..7usize {
                let color = if (main_byte >> bit) & 1 != 0 { WHITE } else { BLACK };
                fb.set_pixel(px as u32, y as u32, color);
                px += 1;
            }
        }
    }
}
```

**Note:** Full DHGR colour uses a 4-bit interleaved scheme (NTSC colour artifacts), but monochrome DHGR (black/white) as above is correct for most text-graphics programs. Colour DHGR can be added later.

## Phase D: Mockingboard Audio

### Task D1: Implement AY-3-8910 PSG

**Files:**
- Create: `crates/apple2/src/ay3_8910.rs`

**Background:** The AY-3-8910 is a 3-voice square wave + noise + envelope PSG. The Mockingboard (Sweet Micro Systems) puts two of them in Apple II slot 4, controlled by a 6522 VIA for each chip. The VIA's Port A connects to the AY data bus, Port B bit 0 controls the AY's BDIR/BC1 lines.

**Step 1: Write the AY-3-8910 implementation:**
```rust
// crates/apple2/src/ay3_8910.rs
// AY-3-8910 Programmable Sound Generator.
// 3 tone channels (A, B, C), 1 noise generator, 1 envelope generator.
// Each channel has a 12-bit period counter, amplitude, and noise/envelope enable.

pub struct Ay3_8910 {
    // Registers $00-$0F
    regs: [u8; 16],

    // Tone counters (12-bit, one per channel)
    tone_counter: [u16; 3],
    tone_state: [bool; 3],

    // Noise
    noise_counter: u16,
    noise_state: u32,   // LFSR
    noise_output: bool,

    // Envelope
    env_counter: u16,
    env_step: u8,
    env_attack: bool,
    env_hold: bool,
    env_alt: bool,
    env_cont: bool,
    env_vol: u8,        // 0-15

    // Audio output
    pub sample_buffer: Vec<f32>,
    sample_rate: f64,
    cpu_freq: f64,
    sample_acc: f64,
}

impl Ay3_8910 {
    pub fn new() -> Self {
        Self {
            regs: [0; 16],
            tone_counter: [0; 3],
            tone_state: [false; 3],
            noise_counter: 0,
            noise_state: 1, // LFSR must not be 0
            noise_output: false,
            env_counter: 0,
            env_step: 0,
            env_attack: false, env_hold: false, env_alt: false, env_cont: false,
            env_vol: 0,
            sample_buffer: Vec::with_capacity(1024),
            sample_rate: 44100.0,
            cpu_freq: 1_023_000.0, // Apple II CPU freq
            sample_acc: 0.0,
        }
    }

    pub fn set_sample_rate(&mut self, rate: u32) {
        self.sample_rate = rate as f64;
    }

    pub fn write_reg(&mut self, reg: u8, val: u8) {
        let r = (reg & 0x0F) as usize;
        self.regs[r] = val;

        // Envelope shape write restarts envelope
        if r == 13 {
            self.env_step = 0;
            self.env_counter = 0;
            let shape = val & 0x0F;
            self.env_attack = shape & 0x04 != 0;
            self.env_alt    = shape & 0x02 != 0;
            self.env_hold   = shape & 0x01 != 0;
            self.env_cont   = shape & 0x08 != 0;
            self.env_vol = if self.env_attack { 0 } else { 15 };
        }
    }

    pub fn read_reg(&self, reg: u8) -> u8 {
        self.regs[(reg & 0x0F) as usize]
    }

    /// Step one CPU cycle.
    pub fn step(&mut self) {
        // Tone channels: period in AY units = (reg_hi:reg_lo) * 8 CPU cycles
        // Each tone flips every `period` half-cycles
        for ch in 0..3 {
            let period = self.tone_period(ch);
            if period == 0 { continue; }
            if self.tone_counter[ch] == 0 {
                self.tone_counter[ch] = period;
                self.tone_state[ch] = !self.tone_state[ch];
            } else {
                self.tone_counter[ch] -= 1;
            }
        }

        // Noise: 5-bit period in regs[6], 17-bit LFSR
        let noise_period = (self.regs[6] & 0x1F) as u16;
        if noise_period > 0 {
            if self.noise_counter == 0 {
                self.noise_counter = noise_period * 8;
                // 17-bit Galois LFSR: tap at bits 3 and 0
                let bit = (self.noise_state ^ (self.noise_state >> 3)) & 1;
                self.noise_state = (self.noise_state >> 1) | (bit << 16);
                self.noise_output = self.noise_state & 1 != 0;
            } else {
                self.noise_counter -= 1;
            }
        }

        // Envelope: period from regs[11:12], 16-bit
        let env_period = (self.regs[12] as u16) << 8 | self.regs[11] as u16;
        if env_period > 0 {
            if self.env_counter == 0 {
                self.env_counter = env_period;
                self.clock_envelope();
            } else {
                self.env_counter -= 1;
            }
        }

        // Sample generation
        self.sample_acc += 1.0;
        if self.sample_acc >= self.cpu_freq / self.sample_rate {
            self.sample_acc -= self.cpu_freq / self.sample_rate;
            self.sample_buffer.push(self.mix());
        }
    }

    fn tone_period(&self, ch: usize) -> u16 {
        let lo = self.regs[ch * 2] as u16;
        let hi = (self.regs[ch * 2 + 1] & 0x0F) as u16;
        ((hi << 8) | lo) * 8
    }

    fn clock_envelope(&mut self) {
        let steps = 16u8;
        self.env_step = self.env_step.wrapping_add(1);
        if self.env_step >= steps {
            self.env_step = 0;
            if !self.env_cont {
                self.env_vol = if self.env_attack { 0 } else { 15 };
            } else if self.env_alt {
                self.env_attack = !self.env_attack;
            }
        }
        self.env_vol = if self.env_attack { self.env_step } else { 15 - self.env_step };
        if self.env_hold && self.env_step == 0 {
            // hold at max/min
        }
    }

    fn mix(&self) -> f32 {
        let mixer = self.regs[7];
        let mut out = 0.0f32;

        for ch in 0..3 {
            let tone_en  = mixer & (1 << ch) == 0;
            let noise_en = mixer & (1 << (ch + 3)) == 0;
            let active = (tone_en && self.tone_state[ch]) || (noise_en && self.noise_output);

            let amp_reg = self.regs[8 + ch];
            let vol = if amp_reg & 0x10 != 0 {
                self.env_vol
            } else {
                amp_reg & 0x0F
            };

            if active {
                out += (vol as f32) / 15.0 / 3.0; // Normalize 3 channels
            }
        }
        out
    }

    pub fn drain_samples(&mut self, out: &mut [f32]) -> usize {
        let n = out.len().min(self.sample_buffer.len());
        out[..n].copy_from_slice(&self.sample_buffer[..n]);
        self.sample_buffer.drain(..n);
        n
    }
}
```

// crates/apple2/src/mockingboard.rs
// Mockingboard (Sweet Micro Systems) in slot 4.
// Two 6522 VIA chips, each controlling one AY-3-8910.
// Slot 4 I/O at $C0C0-$C0CF.
//
// VIA A (addr $C0C0-$C0CF):
//   Port A = AY data bus
//   Port B bit 0 = BC1, bit 1 = BDIR (controls AY command: inactive/read/write/latch)
//
// VIA B (addr $C0D0-$C0DF) — second AY chip, same scheme.

use crate::ay3_8910::Ay3_8910;

#[derive(Default)]
enum AyMode { #[default] Inactive, ReadReg, WriteReg, LatchAddr }

fn ay_mode(bdir: bool, bc1: bool) -> AyMode {
    match (bdir, bc1) {
        (false, false) => AyMode::Inactive,
        (false, true)  => AyMode::ReadReg,
        (true,  false) => AyMode::WriteReg,
        (true,  true)  => AyMode::LatchAddr,
    }
}

pub struct Mockingboard {
    pub ay0: Ay3_8910,
    pub ay1: Ay3_8910,
    // VIA Port B registers for each AY
    via_a_orb: u8,   // controls ay0
    via_b_orb: u8,   // controls ay1
    // AY address latch
    ay0_addr: u8,
    ay1_addr: u8,
}

impl Mockingboard {
    pub fn new() -> Self {
        Self {
            ay0: Ay3_8910::new(),
            ay1: Ay3_8910::new(),
            via_a_orb: 0,
            via_b_orb: 0,
            ay0_addr: 0,
            ay1_addr: 0,
        }
    }

    pub fn set_sample_rate(&mut self, rate: u32) {
        self.ay0.set_sample_rate(rate);
        self.ay1.set_sample_rate(rate);
    }

    /// I/O read from slot 4 space ($C0C0-$C0CF).
    pub fn io_read(&mut self, addr: u16) -> u8 {
        match addr & 0x0F {
            0x00 => { // VIA A Port B — ay0 control
                let bdir = self.via_a_orb & 0x02 != 0;
                let bc1  = self.via_a_orb & 0x01 != 0;
                if let AyMode::ReadReg = ay_mode(bdir, bc1) {
                    self.ay0.read_reg(self.ay0_addr)
                } else {
                    self.via_a_orb
                }
            }
            0x01 => { // VIA A Port A — AY data bus (last written)
                self.ay0.read_reg(self.ay0_addr)
            }
            0x08 => self.via_b_orb, // VIA B Port B
            _ => 0,
        }
    }

    /// I/O write from slot 4 space ($C0C0-$C0CF).
    pub fn io_write(&mut self, addr: u16, val: u8) {
        match addr & 0x0F {
            0x00 => { // VIA A Port B: BC1=bit0, BDIR=bit1
                self.via_a_orb = val;
                let bdir = val & 0x02 != 0;
                let bc1  = val & 0x01 != 0;
                match ay_mode(bdir, bc1) {
                    AyMode::LatchAddr => self.ay0_addr = 0, // addr set via Port A
                    _ => {}
                }
            }
            0x01 => { // VIA A Port A: AY data bus
                let bdir = self.via_a_orb & 0x02 != 0;
                let bc1  = self.via_a_orb & 0x01 != 0;
                match ay_mode(bdir, bc1) {
                    AyMode::LatchAddr => self.ay0_addr = val & 0x0F,
                    AyMode::WriteReg  => self.ay0.write_reg(self.ay0_addr, val),
                    _ => {}
                }
            }
            0x08 => { // VIA B Port B: BC1=bit0, BDIR=bit1 for ay1
                self.via_b_orb = val;
            }
            0x09 => { // VIA B Port A: AY1 data bus
                let bdir = self.via_b_orb & 0x02 != 0;
                let bc1  = self.via_b_orb & 0x01 != 0;
                match ay_mode(bdir, bc1) {
                    AyMode::LatchAddr => self.ay1_addr = val & 0x0F,
                    AyMode::WriteReg  => self.ay1.write_reg(self.ay1_addr, val),
                    _ => {}
                }
            }
            _ => {}
        }
    }

    /// Step both AY chips one CPU cycle.
    pub fn step(&mut self) {
        self.ay0.step();
        self.ay1.step();
    }

    /// Drain mixed audio from both AY chips.
    pub fn drain_samples(&mut self, out: &mut [f32]) -> usize {
        let n = out.len().min(self.ay0.sample_buffer.len().min(self.ay1.sample_buffer.len()));
        for i in 0..n {
            out[i] = (self.ay0.sample_buffer[i] + self.ay1.sample_buffer[i]) * 0.5;
        }
        if n > 0 {
            self.ay0.sample_buffer.drain(..n);
            self.ay1.sample_buffer.drain(..n);
        }
        n
    }
}
